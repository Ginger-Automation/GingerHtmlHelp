Variable Types Overview

1. Dynamic Variables
    This variable can store Dynamic type of data and the value is calculated during Run time.

    Shown in the below pic, select 'Variable Dynamic' and click on the Add Variable button.

    

 

   Double click on the added variable to enter the variable details.

    

2. Variable Selection List
    User can store list of values in this variable. So when Set variable action is used, only the values from the list
    can be stored in this variable. If any other value is tried to be stored then the Action fails.

  This variable is used when user wants to select a default value of his choice.   
    

 

    Double click on the added variable to enter the variable details.

    As shown below, the list of values can be added using the + button.

    The user can select a default value using the 'Selected Value' drop down.

      


3. Variable Password String
    This variable stores the password in encrypted format.
    So the user won't be able to see what password is displayed.

     
    

     Double click on the added variable to enter the variable details.

    After entering the password, click anywhere outside the text box to get the encrypted value.

    


4. Variable Random String
    This variable generates random string during execution.

         

     

    Double click on the added variable to enter the variable details.

    User has to give the minimum and maximum length.

    


    
5. Variable Sequence
   This variable is used when a sequential number is required every time the Business Flow is executed.
   

    

 

    Double click on the added variable to enter the variable details.

    User can give the Min & Max value along with the interval required.

    

6. Variable String
   This is used to store alphabets & numbers. 
   Set Variable Action can also be used to store value in this type of variable.

   

 

  Double click on the added variable to enter the variable details.

  Enter the value in the 'Initial String Value' field.

  

 

7. Variable List   
    User can store list of values in this variable. So when Set variable action is used, only the values from the list
    can be stored in this variable. If any other value is tried to be stored then the Action fails.

       

 

       Double click on the added variable to enter the variable details.

      

8. Variable Random Number
    This variable generates random number during execution.
    

    

    

    Double click on the added variable to enter the variable details.

    User can give the Min & Max value along with the interval required. Integer can also be generated.

    


    
9. Variable Timer
    This variable is used when a timer is required to be set during the execution.

     

 

     Double click on the added variable to enter the variable details.

      


    
        9.1 How to use Variable Timer?
             i)     Click on the ‘Add New’ button in the Activity Actions section.
                    In the Add Action window, select the ‘Set Variable Action’ and click on Add Action button.

              
            
          ii)    Select the variable in the Variable Name drop down. The variable type is automatically populated depending on the variable used. 
                 Select value from the ‘Select Variable Value Operation Type’ drop down.
                 User can start, stop, continue or reset the timer.
                   


          iii)    Click on the Save button
           iv)    When the Action is run, the Timer is started for the Start timer operation type.
                  Timer is shown running in the near the variable name.
