What is an Activity?

An Activity is an area in which a set of Activity Actions can be added in the Automate tab.
Each Activity have a Target application and Actions specific to only that Target application or Non-platform actions can be added.
New Activity can be added as shown in the below screen.

 

 

Click on the Actions Library button to expand it.

 

 

Select the desired Action and click on the 'Add to Actions' button.

 

The Action gets added in the Actions section of the selected Activity.
