Browser Action

This action is used to handle Browser and Widgets. 


How to Configure Action
        1. Pre-requisite 
                  A Solution should be created with Agent having Web, Java or PowerBuilder Platform.
                          
        2. Action Operation- Initialize Browser 
                      
              1)    Double click on the Business flow. In the Activity Actions, click on ‘Add New’ button
                       


              2)    In the ‘Platform Actions’ tab, select the ‘Browse Action’ then click on ‘Add Action’ button given in the below window
                      
                          
              3)    Below screen should be displayed.  
                      The ‘Operation Type’ have default option as ‘Initialize Browser’.
                      Click on Ok to add the action to the Activity Actions.
                      User needs to use ‘Go to URL’ Operation type to launch a website.

                         

 

                    The ‘Operation Type’ for Web application have the below options:-
                 

                    Accept Message Box   -------used to accept a message box which is displayed during execution with a OK button

                    Closee All                      ------- all the tabs opened in a Web browser will be closed

                     Close Tab                     --------the current tab of the web browser will be closed

                    Close Tab Except        ---------all tabs will be closed except the name that will be mentioned in the action

                    Delete All Cookies       --------all the cookies will be deleted

                    Dismiss Message Box  --------used to dismiss a message box during execution which has Cancel button in it

                    Get MessageBox Text   -------to get the text displayed on the message box

                    Get Page Source           -----– used to get source of  a Web page
                    Get Page URL               ----- – used to get the url of the displayed page
                    Inject Java Script            --- – used when we want to inject a different Java script on the web page
                    Maximize Window       -----– used to maximize the displayed window

                    Navigate Back                ------used on a Web Browser to back to the previous page

                    Open URL in New Tab    -----used to open the url in the new tab of the Web browser

                    Refresh                           -------used to Refresh the Web page

                    Run Java Script             -------as said, its used to turn the java script

                    Set AlertBox Text           ------used to display alert message during execution

                    Switch Frame                ------ used when it is required to shift the focus to another frame 
                    Switch to Default Frame --– used when the focus is required to shift to the default frame         
                    Switch to Parent Frame --- - used to switch to the parent frame of the web page
                    Switch Window            ------- used to switch between different windows

                       Wait till page loaded    -------used to wait untill the page is loaded   

 

 

                     The below options of the ‘Operation Type’ are displayed for Java and Power Builder application:-

                        Get Page Source           -----– used to get source of  a page
                    Get Page URL                 ---- – used to get the url of the displayed page

                        Initialize Browser         -------used to initialize a browser when the application has widgets in it

                        Switch Frame                ------ used when it is required to shift the focus to another frame 

                        Switch to Default Frame --– used when the focus is required to shift to the default frame  


            4)    Run Action will perform the action in the Agent.

 

       3.  Action Operation- Handle Browser Alerts

               In the below screen, select the ‘Operation Type’  as per the browser alert required.
               

 

              Below are the list of operation available:-
               1)    Accept Message Box
               2)    Dismiss Message Box
               3)    Get Message Box Text
               4)    Set Alert Box Text


    4. How to use Page Object Model for Goto Url Operation
 
           1.    Open the Browser Action and select operation type as ‘Goto URL’.
           2.    Select the ‘Page Object Model URL’ radio button
                   

          3.    Click on the pen icon and select the Page Object Model from the displayed tree.
                  Click on the Select button of the selection window.

                  
                  

         4.    The select Page Object Model is then displayed. User can modify the page object model if desired using the ‘View/Edit POM’ button as shown below.
