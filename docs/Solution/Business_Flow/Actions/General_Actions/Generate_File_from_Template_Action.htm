Generate File from Template Action

1.    Introduction    
         This action generates a CSV file from a template. 

          
2.    How to create this Action

       1)  Double click on the Business flow under Buisness Flow tab.  Click on ‘Add New’ button and click on Actions Library. 

                                

                         

      2)  Go to the ‘General Actions’ tab and select the ‘Generate File from Template Action’ then click on 'Add to Actions' button shown as below.

                


        3)    Double click on the Action to open it in Edit mode.  Below screen should be displayed.  
               ‘File Action’ is ‘CSVFromTemplate’ option.
                Choose a ‘Data File Name’ using the Browse. Enter name in ‘Data File Name’.
                Click on the 'Run Action' button to execute the action.
