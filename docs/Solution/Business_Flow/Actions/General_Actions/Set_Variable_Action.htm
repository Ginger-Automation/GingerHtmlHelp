Set Variable Action

1.    Introduction    
        Set Variables is an Action which is used to create variables in the solution.
       The below are the variable types:-
          1)    Dynamic
          2)    Selection List
          3)    Password String
          4)    Random String
          5)    Sequence
          6)    String
          7)    List
          8)    Random Number
         
2.    How to configure this Action
       1)    Go to Automate tab. expand the Variables section and click on ‘Add New’ button
               

       
      2)    The ‘Add Variable’ window should be displayed. Select the desired variable type and 
             click on ‘Add Variable’ button.

              
  
      3)    Below ‘Edit VariableDynamic’  screen should be displayed. 
              Enter the Name. Enter the description and Value expression  if required.
            ‘Set as Input Value’ is checked by default.
             Linked Variable can be used when another variable is already created.
             Few variables can have initial value that can be entered in ‘Initial String Value’ test field which gets displayed as the ‘Current value’
            Click on Ok button to add the Variable in the list.

            
                      
     4)    The action gets added in Activity Action list

             

                
     5)    This Variable can now be used in the Actions. Expand the ‘Activity Actions’ area and click on the ‘Add New’ button. 

            
 
     6)    The below screen is displayed. Go to ‘General Actions’ and select the ‘Set Variable Action’ and click on Add Action button.

            
 
     7)    Below screen is displayed. 
             The ‘Variable Name’ drop down displayed the list of variables that are added in the ‘Variables’ area.
           ‘Variable Type’ displayed the type of variable that is selected.
            ‘Set Variable Value Operation Type’ drop down displayed the action that is required to be done on the Variable. The options are                       ‘AutoGenerateValue’, ‘SetValue’  and ‘Reset Value’

            
 
    8)    Click on Ok button adds the Action to the Activity Actions
