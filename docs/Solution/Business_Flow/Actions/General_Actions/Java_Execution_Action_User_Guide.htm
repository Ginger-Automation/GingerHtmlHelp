Java Execution Action

Description    
    The “Java Integration Action” should be used in case a Java executer need to be integrated in the automation flow.
    Using this action user is able to run the Java Jar with set of inputs sent from Ginger and process the returned values of the Jar.
    
How to Use It
  1) Setup the Java Executable (Jar)
    The Java executable need to be adjust to work with Ginger.
    Please find below the items which needs to be configured in it Main class:
        1.    Support for “-GingerHelp” command line arguments- when the Jar been executed using this argument then it supposed to write to                   Console the Java executable description + the required parameters for execution
        2.    When the Java executable finish to do the work it meant to then it should write to Console the output values which Ginger require                   to show/parse inside “~~~GINGER_RC_START~~~”/“~~~GINGER_RC_END~~~”.
