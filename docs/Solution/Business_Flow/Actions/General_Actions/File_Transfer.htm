File Transfer

1)    Go to Automate tab. In the Activity Actions, click on ‘Add New’ button
               

2)    Go to the ‘General Actions’ tab and select the ‘File Transfer Action’ then click on ‘Add Action’ button given in the below window
                  

3)    Below screen should be displayed.  
       Select the desired file using the Browse button near the ‘PC Path’.
       Enter the Unix path, host, port, username, password, PrivateKeyFile and KeyPassPhrase (If any).
       Click on Ok to Add the action to the Activity Actions.
