How to use Cassandra in Ginger?

1. What is Cassandra? 

   Apache Cassandra is a free and open source distributed NoSQL database management system designed to handle large amounts of data 
   across many commodity servers, providing high availability with no single point failure.

2. What does Ginger support in Cassandra? 

    a) UDT values which has been widely used in Cassandra Database.
    b) JSON variant of SELECT and INSERT operation

3. Which Cassandra operations are supported in Ginger? 

    a)    Read Data
    b)    Write Data
    c)    Update Data
    d)    Record Count

4. How to configure Cassandra in Ginger? 

    a)    Under the Resources tab, go to Environment and select the desired environment. 
    b)    Using the + button add the Cassandra DB. Enter name, select DB Type as “Cassandra”, enter the host and user name and password if any.

              

    c)    Click on the Test connection button to get the connection successful message.
            

          
 

5. How to add Database Action for Cassandra? 


    a)    Double click on the Business flow under Buisness Flow tab.  Click on ‘Add New’ button and click on Actions Library.
               
              

    b)    Go to the ‘General Actions’ tab and select the ‘DataBase Action’ then click on ‘Add to Actions’ button shown as below.
              


    c)    Double click on the Action to open it in Edit mode.. Select the environment and the DB and the operation type.
           The DB operation type as same as given in the '4. Action Operation' section of topic 'How to Configure Database Action?' .

            

    d)    Run the Action to execute it
