Clear All the Variables Action

Introduction    
Clear All Variables is an Action which is used before starting the Solution suite to clear all the variables to make them blank before using them. 
When a solution is run and values are stored in the variables, these values can create issue when the solution is run the next time. To avoid such issues this Action is created.

The below variable types values are made blank:-
1)    Selection list
2)    String

Pre-requisite 
    Before performing this Action a set of variables is required to be created in the Solution.
    The below section can be referred from Ginger Help Library to know how to create variables.

    Solution-> Business Flows-> Activities-> Actions-> Actions Types Overview-> 
    General Actions-> Set Variable Action

How to Configure Action
           
   1)   Double click on the Business flow under Buisness Flow tab.  Click on ‘Add New’ button and click on Actions Library.
               

   2)    Go to the ‘General Actions’ tab and select the ‘Clear All Variables Action’ then click on Add Action button shown as below.
               

               


  3)    The action gets added in Activity Action list. Double click on the Action to open it in Edit mode.
