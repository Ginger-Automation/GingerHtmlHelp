WebAPI REST Action

How to Configure Action
    1. Pre-requisite 
         A Solution should be created with Agent having Web Services Platform.
                          
    2. Action Operation
                      
         1)    Go to Automate tab. In the Activity Actions, click on ‘Add New’ button
                   


         2)    In the ‘Platform Actions’ tab, select the ‘REST Action’ then click on ‘Add Action’ button given in the below window
                                 

 
         3)    Below screen should be displayed.  
                  •    End point url needs to be given
                  •    The ‘Request Type’ has multiple options out of which the important one is ‘Get’ and ‘Post’
                  •    ‘Response Content Type’ has 3 options viz. json, utf-8 & xml.  Option other than these can also be used by writing into it.
                  •    Cookies can be used if required
                  •    Click on Ok to add the action to the Activity Actions.

                          

                        
                  •    The Request Body is mandatory for ‘POST’ request type. Parameters can also be added here.
                            


                  •    The Security Type can be added if required alone with the Authorization details.
