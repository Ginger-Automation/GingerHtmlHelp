Web Agent

i)    Open the Add New Agent Wizard and click on the Next button.
      Enter the name, platform as Web and Driver type as per the requirement.
      Make sure that the required deriver type is available on your PC.
      Eg. If you are selecting Firefox then latest Firefox version should be installed on your PC.

      Click on Add button to associate a Tag if required. Click on the Next button.

       

 

      The below wizard is displayed. It gives information about the Agent and its usage.  Click on the Next button to add a new agent.

     
       

      

      In the below window, enter the Agent name. Select the Platform type as Web. Driver Type as Internal Browser is populated by default.

      

       The  Driver Types are IE, Chrome, Firefox, Remote Web Driver & Edge.

ii)    User can enter any Proxy if required in the Driver Configuration page.
       Click on ‘Test Agent Configuration’ button. Connection successful message is display if connection is a success.

       Click on the Finish button to add the agent.
