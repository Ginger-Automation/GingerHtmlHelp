Internet Explorer Pre-Settings for Application

1.    When to use the Internet Explorer Pre-Settings?    
       When Internet Explorer is to be used as a Target application then certain mandatory Internet Explorer pre-setting are required to

       be done so that the Internet Explorer Agent runs smoothly in Ginger.

 

2.    What are the IE Pre-Settings?
               
    •  Open internet options, Enable protected mode for all security zones.

           

 

           

 

          

 

         

 

                            
                
    •    Uncheck the pop up blocker from privacy settings. 

          This setting is required when you want to perform Switch tab or Switch window action.
