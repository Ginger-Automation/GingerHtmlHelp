Run Web Agent in Hidden mode

 

           This functionality is available only for the Chrome and Firefox Agents.

 

           When the Agent is run (either from Automate tab or the Runset tab), it is not visible.

           Only its console is visible.

 

           To do this, select the required Agent. Click on theDriver Configuration.

           The parameter ‘HeadlessBrowserMode’ has to be made as ‘true’. By default it is marked as false.

           This mode is used for performing the execution faster.
