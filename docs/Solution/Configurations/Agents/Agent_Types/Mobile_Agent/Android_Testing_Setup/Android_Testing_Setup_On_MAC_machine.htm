Android Testing Setup On MAC machine 

 

These steps will allows Ginger to run tests on an USB connected Android device using Appium.

 Prerequisites
    i)    Ginger need to be installed on a Windows operating system machine.

    ii)    The server can be installed on  Mac machine (Mac Mini, Mac Book, etc.) with OS X operating system (OS X Yosemite or later) is

           required

         1.    Mac: Install Appium Prerequisites

                1.1    Install latest Java JDK:

                           a.    Download the latest Java JDK version from below link:

                                  http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html
                                       


                           b.    Double click and install it

 

                 1.2    Install Homebrew:
                           a.    Go to the Mac main menu bar -> “Utilities” -> “Terminal”
                           b.    Type and run below command:
                                  ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
                           c.    Make sure installation ended successfully
                                  Note: for allowing network access from the Terminal you may need an external network connection (non Amdocs) only for the installation purpose


                  1.3    Install Node:
                            a.    Go to the Mac main menu bar -> “Utilities” -> “Terminal”
                            b.    Type and run below command:
                                    brew install node
                            c.    Make sure installation ended successfully


                  1.4    Install Android SDK Tools:
                            a.    Go to https://developer.android.com/sdk/index.html
                            b.    Click on Other Download Options
                                    


                            c.    Download the SDK Tools Only which fits to Mac OS X

                                         

 

                            d.    Once it was downloaded and put it in desired place. Mark it full path, like in this below case it :

                                   “/Users/menik/android-sdk-macosx”

                                     

                 1.5    Set the Environment Variables:
                            a.    Go to the Mac main menu bar -> “Utilities” -> “Terminal”
                            b.    Type and run below command:
                                   nano ~/.bash_profile
                            c.    Editor window will open in Terminal
                                         

 

                            d.    Copy and paste to it below content
                                   export ANDROID_HOME=<<path_you_found_in_step_1.4.4.d>>
                                   export PATH=$ANDROID_HOME/platform-tools:$PATH
                                   export PATH=$ANDROID_HOME/tools:$PATH
                                   export JAVA_HOME=$(/usr/libexec/java_home -v <<your_Java_version_like_“1.8”>>)
                                   export PATH=/usr/local/bin:$PATH

                                   


                           e.    Click CTRL+O and then Enter in order to save this file
                           f.    Click CTRL+X in order to close it
                           g.    Open Terminal again (if it was closed) and type below command to apply the environment variables values:
                                   source ~/.bash_profile
                           h.    Check that the new environment variables were saved by typing below command in Terminal and looking for the

                                  relevant variables and their value:

                            a.    Go to the Mac main menu bar -> “Utilities” -> “Terminal”
                            b.    Type and run below command:
                                     android
                            c.    Android SDK Manager will be opened
                            d.    Mark below 3 Packages to be installed and install them
                                              

                                      

                                         

 

                            e.    Make sure the installation ended successfully

                                   


          2.    Mac: Install Appium 

                       a.    Open the Appium Web site on the Mac machine (using Safari)
                               http://appium.io/
                        b.    Click on Download Appium button
                                


                        c.    Double click and install Appium application
                        d.    Launch the application from main menu bar “Go” “Applications”
                                 


                        e.    For checking that noting is missing, click on the “doctor” button in the Appium GUI
                        f.    Terminal window supposed to open and tests will be done for iOS and Android setup, make sure the Android one passed

                               successfully (iOS will be installed in different user guide)
                              Note: in case the Appium doctor (or on Appium launch) identify that something is missing and asks you if to install

                                         it- answer YES

                              


           3.    Mac: Configure and Launch Appium Server 

                       a.    Connect the Mac machine to the same network that Ginger Windows machine is in
                       b.    Get the Mac IP Address by:
                       c.    Go to main menu Go-> Applications-> System Preferences-> Network
                               


                       d.    Copy the IP Address
                               


                        e.    Launch (if not already launched) the Appium application from main menu bar Go-> Applications-> Appium
                        f.    Click on General Settings button
                       g.    Set the IP Address you found in step b into the Server Address field
                       h.    Configure the rest of settings as appear in below screen shot
                                 


                        i.    Close the General Settings window
                        j.    Click on Launch button
                               


                       k.    Make sure appium is starting without errors

                              
 
                        l.    Appium server is UP and ready to get connecting request from Ginger.
