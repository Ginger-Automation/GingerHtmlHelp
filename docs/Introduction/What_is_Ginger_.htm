What is Ginger?

Introduction              

Ginger is the NextGen Amdocs Automation Tool used to automate the testing of various applications.

 

It is free to use and enhance by the open source community.

 

It provides a unique platform for automation with a wide range of drivers that support a number of OS and applications/technologies.

The Platforms are Web, Android device, Mobile, Main Frame, Power Builder, Java, Unix, VB Script and Windows.

 

It has AI driven Automation providing Auto pilot functionality which will automatically learn tested application capabilities and model them.

 

It has Data driven automation from multiple sources (Excel, Database etc.).

 

It has Powerful, elaborate and professional-grade reports via email and browser.

 

It enables command line execution  with Jenkins integration.

 

It has Integration with leading Application Lifecycle Management tools.
