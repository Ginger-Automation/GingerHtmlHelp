Gherkin – TextEdtor for .Feature Files

The integration of Gherkin into Ginger using the Gherkin TextEditor allows users to easily read, write and review Gherkin Feature files in Ginger.


1.    Feature TextEditor – Layout
        The Gherkin TextEditor is fully integrating in Ginger.
         TextEditor Layout
         You can see how the Gherkin TextEditor fits into Ginger and how its layed out in the following Fig 1.1.
           A.    This is the Current Feature file that you are working on.
           B.    This is the Feature name on the first line, and the description in the following lines BEFORE the “space line”
           C.    This is the Scenario with following Keywords that make up the Scenarios Steps
           D.    These are Steps that are within the Scenario
           E.    These are Tags that you can associate functionality within the Feature
           F.    These are the existing automation steps that are already in Ginger Shared Repository having at least one tag associated with the feature or Scenario.
