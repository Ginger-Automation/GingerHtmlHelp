How to use Ginger Website?

The Ginger Website is created for allowing Ginger users to configure and trigger Ginger Executions from any machine without Installing Ginger.
Ginger Website offers:
    -    Create a Job to create and schedule Ginger Run Set Execution
    
Pre-Requisites
  1. Ginger Solution which:
        a.    Configured on Ginger SVN/GIT Data Source
        b.    Includes Run Set/s ready to be executed 
  2. Jenkins Server version 1.606 or above with Ginger Plug 1.5 or above Installed
  3. Jenkins Slave installed on the machine hosting Ginger(Beta 2.5.9 or above) installation (machine also need to be able to access SVN server in which the Ginger Solution exist)

How to Generate API Token
    •    Login to Jenkins.
          


    •    Go to Configuration.
           


    •    Click on Show API Token.
          

             
    •    Copy the API Token
