1. How to Create a Job?

Create Job Page allows you to create a new job to trigger/schedule execution for a particular run set.
Job Configuration Mandatory Parameters:

Parameter Name

Explanation

Title

Name of the Job

Executor

Jenkins Node name (connected on slave machine). Refer How to connect Slave

Runset

Runset to be executed

Environment

Environment on which execution will be triggered

Job Configuration Advanced Parameters:

Parameter Name

Explanation

Enabled

Job Enabled/Not

Ginger Installation Path

Path where Ginger is installed on the executor machine

Remote Solution Path

File system Path where the solution will be downloaded

Build Period

Refer Configure Build Period for valid values
