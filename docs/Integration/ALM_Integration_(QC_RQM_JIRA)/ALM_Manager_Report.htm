ALM Manager Report

This report can be used for getting below information for each ALM TC which included in the selected Tests:

  1. Automation coverage % by Ginger

Note: automation coverage will be reflected according to the current loaded Ginger solution (based on it Shared Repository

  1. Ginger Executions details (which were exported to ALM):
    1. Number of Executions
    2. Pass Rate
    3. Last execution

User is able to filter the execution details by executing tester and/or execution period (last day, last month, etc.)

For using the report please follow the below:

  1. Connect to ALM (see section 3)
  2. Click on “ALM”à“ “ALM Manager Report” from the main menu drop down
  3. Click on “Browse” button to select the Test Set or Test Sets folder you want to get info about from the ALM Test Plan tree

          

 

          

  1. View the “Coverage” details

          

  1. For shifting to “Execution” view select “Execution” from the View drop down
  2. Use the “User” and “Execution Period” filters to filter the shown data
