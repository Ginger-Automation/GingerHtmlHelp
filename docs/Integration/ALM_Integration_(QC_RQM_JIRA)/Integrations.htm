Integration

Amdocs BEAT Ginger application was enhanced to integrate with ALM (QC and RQM) application and this document comes to explain how to use the current Ginger-ALM functionalities.

Current supported Ginger-ALM functionalities are:

  • Import ALM Test Set into Ginger and automatically converting it to Ginger Test Set which will be ready to be automated or even execute in case it’s TC’s were already automated before by Ginger

 

  • Export Ginger Business Flow To ALM- each one of the Business Flow Activities Group will be export to ALM Test Plan as Test Case item and a Test Set item will be created in ALM Test Lab containing all the relevant ALM Test Cases which were exported

 

  • Export imported Ginger Test Set execution details back to ALM and reflect the execution status in the matching ALM Test Set TC’s Run status

 

  • Review ALM Test Cases automation coverage and executions statistics done by Ginger using the “ALM Manager Report”
