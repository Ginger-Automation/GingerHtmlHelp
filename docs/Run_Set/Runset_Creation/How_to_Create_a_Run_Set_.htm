How to Create a Run Set?

A Run Set is a set of selected Business flows which are grouped together for execution.

This is very powerful feature for parallel execution.         

 

                A Business Flow has to be created in order to utilize the Run set features.

 

2.1 Default Runset

 

                   When a solution is created a default run set is automatically added to it.           

                    A Runner, Runner1 is also available by default and is highlighted.

                   Business Flows need to be added which is covered in How to Add Business Flow in Runner?

                      Under the Run tab, click on the + button as shown below at the extreme top right.

                      

                       In the new window, enter the Runset name and click on the Ok button.

                      

                     

                       A new Runset is added with a default Runner in it.
