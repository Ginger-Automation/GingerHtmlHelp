Run Set Extra Functionalities

1.    How to Run Analyzer?

    a)    The Analyzer Run set button is displayed as shown below in the drop down.                                                                         
                              

    b)    Below Analyzer screen is displayed. It gives details of the Business Flow that has an issue along with the description of the issue, the             status, if  it can be auto fixed  and the severity of it.

                                              


    c)    If the ‘Can Auto Fix’ is Yes then click on the ‘Fix Selected’ button and the status changes to Fixed. Click on the ‘Save Fixed Items’               button to save the changes in the Solution.
           If the issue cannot be fixed automatically then the user needs to manually fix the issue before execution.
           User can also cross check the Auto fix if it is as per the user expectation. 

            


     
    d)    Click on the Re-Analyze button to check the fix. The issue is not populated.

                                            


2.    How to disable the Analyzer?
               
        The Analyzer can be disabled from the Configurations tab.
         Uncheck the checkbox as shown below and execute the Run set. The Analyzer will not be called and run set will be executed.

4.    How to reload Run set?

      When a user works on the configurations of a Run set and he wants to restore the Run set to previously saved state i.e. he does not want        to save his new changes, then he can use the below button to reload the Runset.

     

      

    Click on the Yes, the Run set will be loaded to the state where it was last saved.
