How to create Run Set Shortcut for Command Line?

1. Go to the Run tab of the solution

    Click on the ‘Create Run Set Auto run Configuration’ button as shown below.

                     

 

    The Auto Run configuration Wizard is opened giving information of about.. Click on the Next button.   

       

              

 2.  Choose the desired option and Click on Next button. Refer No.1 section of "How to create Run Set's Auto Run Configuration?" for more details.

                                  

                    

          

  3. Keep the default config file selection. Use Dynamic file if you want to make changes in the parameters or want to create a new Run set without opening Ginger.

       Refer section 2.2 of "How to create Run Set's Auto Run Configuration?" for more details.

        Click on the Next button.

                  

 

  4.  Select the shortcut folder and click on the Finish button.

         

                      

   5.  Right click on the Short cut and Copy the Target field.

                       

 

         Go the Command line prompt and paste it here.

         Press Enter to execute the short cut.
