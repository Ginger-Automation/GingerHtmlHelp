How to use Global Parameter in Value Expression?

 

  The “Current Value” of a global parameter is the value which will be set for the VE expression in run time.
  The global parameter can be used in other actions also using the Value Expression window.
  In the any Action click on the ‘…’ as shown below. 

     

               

  In the Value Expression window, the global parameters is displayed in the left panel. Select the desired and 
  double click on it to get it into the editor pane.
