How to De-link item from shared repository source?

Note:-Only those items can be de-linked which were added from the shared repository.  

        The items that were used to create shared repository cannot be de-linked.

1.    Select the desired item and open in Edit window. Click on the star icon.

            


2.    Below confirmation message is displayed.
        

3.    Click on Yes button. The item is de-linked from shared repository and the star color changes to grey.
