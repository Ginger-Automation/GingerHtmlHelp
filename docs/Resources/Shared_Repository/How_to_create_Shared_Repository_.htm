How to create Shared Repository?

1.  How to create Activities shared repository?
        a.    Double click on Business Flow in the Business Flows tab.
        b.    Go to Activities Section and click on the Star button as shown below.
               Note: Activities section is present in both Automation and Solution Tabs.

         •    All activities will be listed down for that business flow. Click on the Star+ button (shown in red arrow below) to add                                         selected activity to a repository.
         •     Note that the star icon (shown in green arrow below) is grayed out as the Activity is not yet available in Shared Repository
             
             
        c.    Once the Activity is added to the Shared Repository, the star icon near the Activity in the Activities section turns to Orange color 

               

        d.    Activity is added in the Activities section of the Shared Repository
                           

        e.    The view can be changed to tree view using the button shown below.

                 

          
2.    How to create Actions Shared Repository?

        a. Double click on the desired Business Flow from Solution tab.
        b. Business Flow details will get open in Automate tab. Go to Actions Section.

          •   See the Actions window in below image of an Activity.

          •   All Actions will be listed down for the Activity. Click on the Star+ button (shown in red arrow below) to add selected 
              Action to the repository.

              Note :- the star icon (shown in green arrow below) is grayed out as the Action is not yet available in Shared Repository

               
        c.    After the action is added in the Actions section of the Shared Repository, the star turns orange is color.

               

        d.    Activity is added in the Actions section of the Shared Repository              

                                           
                               
3.    How to create Variables Repository?

        a.    Double click on the desired Business Flow from Solution tab. 
        b.    Business Flow details will get open in Automate tab. Go to Variables Section.
        c.    See the Variables window in below image.
               Note: Variables section is present in both Automation and Solution Tabs.

        d.    All Variables will be listed down for the Business Flow. Click on the Star+ button (shown in red arrow below) to add selected                         Variable to the repository.

            Note:- the star icon (shown in green arrow below) is grayed out as the Variable is not yet available in Shared Repository

                
                         
        e.    After the action is added in the Actions section of the Shared Repository, the star turns orange is color.
                             

        f.    It gets added in the Variables section of the Shared Repository.
                                  
                     
4.    How to create Activities Groups Repository?

        a.    Double click on Business Flow under Test from Solution tab. 
        b.    Business Flow details will get open in Automate tab. Go to Activities Groups Section.
            •     See the Activities Groups window in below image.
                  Note: Activities Groups section is present in both Automation and Solution Tabs.

            •     All Activities Groups will be listed down for the Business Flow. Click on the Star+ button (shown in red arrow below) to add                         selected Variable to the repository.

                  Note:-the star icon (shown in green arrow below) is grayed out as the Activity Group is not yet available in Shared Repository

                  

  
        c.    After the action is added in the Actions section of the Shared Repository, the star turns orange is color.
                                      

        d.    It gets added in the Activities Groups section of the Shared Repository.
                   

                  
        e.    The view can be changed to tree view using the button shown below.
