How to Add New Ginger Data Source & Table

Default Ginger Data Source & LiteDB are added to every new solution.

The MDB & LiteDB files are copied to the Data Sources Folder of the Solution in the File System. 

 

1.    Go to Resources-> Data Sources tab.
       Right click on the “Data Sources” node and Select “Add Data Source” from the Pop Up Menu.
        


2.    There are 2 Data Source Types viz. “MS Access” and "LiteDataBase".

       a) MS Access
            Add the Data Source Name and the Path of the external MDB File. Click OK

             

 

      b) LiteDataBase

          It is an open source MongoDB-like database with zero configuration

           Add the Data Source Name and the Path of the external LiteDB File. Click OK

           

 

3.    Add New Table

         There are 2 options available viz. Customized table and Key Value Table.

           To add a Table to the Data Source, select the desired option.

                   


                 Enter the Table Name and click on the Ok button.
