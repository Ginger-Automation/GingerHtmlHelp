What are Ginger Plugins?

Plugins are add-ons that adds specific feature to ginger, we are using Plugins to enable third party developers
To create abilities which extend ginger and to easily adding new features.  

1.    Adding new Plugin
          a.    Go to Resources->Plugins. Click on 'Add Item' button as shown below.
                  


         b.    In the wizard, click on Next button

                   

            c. Browser the folder and click Next button

                     

           d.   On this page you will be able to see all the Information about the plugin 
                 1.    plugin package id 
                 2.    Folder – relative path for embedded plugin and full Path for system plugin.
                 3.    Version -   embedded/ system.
                 4.    Actions – list of all available action on this plugin  
                 5.    Text Editors – if plugin supports text editors , it will be listed on the text Editors tab.
