How to sending Input Values to TestNG Flow from Ginger Flow?

TestNG plugin supports sending input values from Ginger flow to TestNG flow.
It been done by overwriting the values of the TestNG XML Parameters.
Input values can be static or dynamic (variables, environments parameters, Etc.) by using the Value Expression capabilities.
In case Input values been configured, the plugin will duplicate the original TestNG XML into temp file, will overwrite the Parameters values on the duplicated XML and will 
use the duplicated XML for actual execution.
In case user prefer to overwrite the original TestNG XML and not to create temp duplication he should check the “Overwrite Original Testng XML” check box. 

 How to Send Input Values to TestNG Flow from Ginger Flow?
    1.    Configure any one of the TestNG Actions configurations
    2.    Set the “Testng XML Path” field in the Action configurations
    3.    Open the TestNG XML to view it content and look for the “Parameters” nodes in it which you want to overwrite it values
    4.    For each parameter which you want to overwrite it value:
            a.   Add row to the “Testng Xml Parameters To Overwrite” grid
            b.  Set the cell of the “Name” column as the name on Parameter in the TestNG XML
            c.  In case the Parameter appears in more than one node and you want to modify only specific one and not all so set the name on the                   Parameter Parent node in the cell of “ParentNodeName” column.
           d.  Set the value to be used in the “Value” column cell
               All cells values can be fully/parsially dynamic using Value Expression editor (“…” button).
