How to Export to Java Or Jason File?

1. How to Export to a Java File?
    Pre-requisites: 
     •    Pact PlugIn loaded.
     •    Pact File Created or Imported to the Documents Folder.
     •    Pact File Compiled.


    1.    On “PACT Content Editor” Page Click on Export To Java button
           


    2.    It will ask you if you want to use the default Java template. 
             


        a.    When user clicks on Yes, Ginger will take the ordinary Java template and display the folder where the files are stored.

              


               
        b.    When user clicks on No, you will be asked to select a new template file, make sure the file you are selecting contains the Interactions Data place holder “<<Interactions_Data>>” 
            otherwise the export will be failed.
    4.    Once the Export process will finish the selected folder will be opened and the file will be inside with the following format “PactToJava<<TimeStamp>>.Java” for example 
        “PactToJava13_11_2017_17_13_47.Java”
    5.    Also a success message will be popped up

 

2.    How to Export to a Json File?
    Pre-requisites: 
     •    Pact PlugIn loaded.
     •    Pact File Created or Imported to the Documents Folder.
     •    Pact File Compiled.


    1.    On “PACT Content Editor” Page Click on Export To JSON button
           

    2.    Select a folder to export the JSON file to
    3.    Once the Export process will finish the selected folder will be opened and the file will be inside another folder with the following format “PactToJSON<<TimeStamp>>” 
        for example “PactToJson13_11_2017_17_18_02.Java”
        Inside this folder you will see the json file with a naming convention which built from the Consumer and Provider defined on the PACT Content Editor.
    4.    Also you will find a log directory for this extract execution tracing 
    5.    A success message will be popped up.
