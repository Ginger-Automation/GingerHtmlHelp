How to Create Or Import PACT Document?

1.    How to Create Pact Document?
    Pre-requisites: 
    Pact PlugIn should be loaded.


        1.    From the left tree menu right click on “Documents” folder 
        2.    Select “Created Document” -> “Add Other File Type”
        
             

             Note:- if you can’t find “Create Pact Document” option you probably haven’t loaded the plugin as expected

        3.    Enter Pact file name along with the extension ".pact" and click on Ok button

               


        4.    Pact file is created. Click on it to see the Pact Content Editor.

                Either type the contents in the file or copy paste it here from another file.  

               


2.   How to Import Pact Document?
    Pre-requisites: 
    Pact PlugIn should be loaded.


    1.    From the left tree menu right click on “Documents” folder 
    2.    Select “Open Folder in File Explorer”

           
 

    3.    You can copy paste the desired file in this folder.

          

     4.    Right click on “Documents” folder in Ginger and select the "Refresh" option.

               


    5.    The copied Pact file is displayed. Click on it to see the Pact Content Editor
