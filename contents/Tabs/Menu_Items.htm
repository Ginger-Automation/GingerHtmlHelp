Menu Items

1.    Drop Down    

              
    

        Click on the drop-down displayed on the Solution name.
        As shown above, user can Open recently used solution, existing solution and create a 
        new solution.
        Download existing solution is used when the solution exists in the SVN  server.


2.    Find and Replace    

           

         This button is used to find and replace the automation items like Business Flow/ Activity /Action /Variable

 

3.    Share Operations    

             
               

        This is used to share the solution with ALM.
            a)     After giving the ALM connection details following activities can be done:-
                       -    Import test case from ALM to automate them in Ginger
                       -    Export Activity Groups to ALM
                       -    Export execution results to ALM
            b)    ALM Items Fields Configuration:- is used to give values to the fields in ALM that are required for opening a defect.

            c)    ALM Defect’s profile:- this is the area where the defects name and other details can be given so that Ginger can open

                                                           defect automatically in ALM


4.    Extra Solution Operations    

              

        Click on the Extra Solution Operations drop down to get the above features.
            a)    Analyze Solution:- this helps to find problems and issues within your solution\Business Flow\Run Set. 
                                                    Problems that will prevent your flow from running correctly or getting failed. 
                                                    This problems can be defined as wrong implementation.
                       
            b)    Upgrade Solution:- this is used when a solution from older Ginger version needs to be upgraded to latest version.

            c)    Recover Solution:- in case Ginger is shut down unexpectedly, then the updates of the solution can be recovered using                                                             this option.
            d)    View/ Edit Solution Details:- 
            e)    Open Solution Folder:-this opens the solution in File Folder structure

 

5.    User Profile    

             
                

            a)    User Profile    :- here the user can update his profile like name, email id etc.
            b)    User Settings :- settings related to notifications of upgrade, recovery etc are displayed here
            c)    Log                   :- Ginger Log related options are available here 


6.    Extra Options
               

           a)    Ginger Help Library :- A list of support documents and videos of Ginger are displayed here. 
           b)    Support                      :- Creating a ticket, Q&A forum and opening support site can be done using this option.

           c)    Contact                       :- this gives the email distribution list to contact the Ginger Support team and the Ginger Core team.
           d)    About                          :-  this gives details on the version Ginger
           e)    Close Application      :- as per the name this closes the application
