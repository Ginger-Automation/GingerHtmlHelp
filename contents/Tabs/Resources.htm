Resources

 

  1. Shared Repository

          The Activities/Actions/Activity Groups/Variables added into the Shared Repository from the Business Flows Tab is displayed                 here. User can make updates to the items if required.

              

  1. Application Models       

After adding the APIs in this tab, they can be used in the Web API Model Action on the

Automate tab for the Web Services applications.

For more details refer to the ‘Application Models’ document under ’06-Resources’

section.

   

         

  1. Environments      

This is the tab where the environment are displayed. There is a Default environment automatically created when a new solution is created. Use can add multiple environments if required.

  

 

  1. Global Variables  

          Variables that has to be used in all the Business Flows of the solution can be added in this tab.

                

      

  1. Data Sources       

           Ginger has an internal data source which acts as a common place to Read/Write data in all Business Flows/Activities/Actions.

            There is a Key_Table and a Customized table provided.

             

                                          

  1. Documents

          All the documents in the solution are displayed at the below location.

          Folders are created at the time of solution creation for better organization.

             

  1. Plugins       

          Plugins required in the solution like the PACT are be added from this place.
