Configurations

1.    Target Applications    
        This is the application on which the execution is going to be done.
        User can add multiple Target Applications in a solution from this tab.

       

 

2.    Agents    
       Agents are the drivers which are used for execution for the target application under test.
       A solution can have multiple agents in it.

       

 

3.    Reports    
       After a Business Flow is executed, results are generated as per the configuration done in the default Report Template
       

 

4.    Tags
        Tags are created from this tab. They are linked to multiple Automation items like Actions/Activities/API Model in order to                     identify them to set of functionalities.
