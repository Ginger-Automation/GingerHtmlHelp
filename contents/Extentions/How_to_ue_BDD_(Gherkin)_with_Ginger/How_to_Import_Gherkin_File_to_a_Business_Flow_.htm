How to Import Gherkin File to a Business Flow?


The Ginger Automation Tool can import any Gherkin <file>.feature into Ginger and convert them to a Automation Business Flow in Ginger.


1.    IMPORT – Gherkin .feature file to BF
    In the Solutions Tab, under Business Flows (we suggest to organize with folders like in Fig 1.1), Import Instructions.
        •    Right click on Business Flow or on the Folder you want to create the BF in, and in the dropdown select “Import” -> “Import Gherkin Feature”. Fig 1.1

        •    The File selection window will show, select “…” to browse windows location for your .feature file and click “Open”. Fig 1.2

        •    You will be able to review the Gherkin file in the preview before selecting “Import”. Fig 1.3 ->then select “Generate Business Flow”
           
           


          Click on the Next button in the below wizard. 

          

 

        Click on the "..." button as shown below to select the desired file.

 
        

 

        Click on the Next button.
     
               

       

          User can create a sub folder inside the Business Flows by right clicking on it & selecting 'Add Sub Folder' option.

          

 

          Click on the Finish button.

          

 

         A success message is displayed and Business Flow created message is displayed.

           

 

           The file is displayed as shown below.

           Fig. 1.1

           

 

          The Activity Groups added are in de-active status.  Click on the button as shown below to Activate them.

          

 

         Double click on the Business Flow. The details of the Feature file can be seen using the button as shown below.

           


2.    IMPORT – Gherkin Business Flow creation from .Feature file 
        After importing the Gherkin .feature:
        Generate Business Flow instructions from Gherkin File      

          •    To create a Business Flow,  select the feature file in the Documents folder. Go to the Optimization Results tab and click “Generate Business Flow” button.

                Fig. 2.1

                

          •    If you see “Update Business Flow” then the Business Flow already exists, and you can update the existing flow.
               This will create a business flow INSIDE the folder that you originally selected during the import. 

                   

 

                      Fig.2.2

                             
       
                 
  

3.    IMPORT – Gherkin Optimized Activities to Automation Scenarios in Ginger
         If you are creating your scenarios using Gingers Gherkin TextEditor; 
        After you have completed your new .feature:
       Optimized Activities
        You learned in section 1 – 2 how to compile and view the optimized activities in Fig 1.1 &  Fig. 2.1   

         LAYOUT – Gherkin “Optimize” to Automation' section of "What is Gherkin?" topic.
        If your Business Flow does not exist you will see in the optimize tab “Create Business Flow”. Fig 2.1
        This will create your Gherkin Business Flow as shown in Fig 2.2.
