How to Import optional values for parameters offline?

Select the desired API and go to the Model Parameters tab.
Click on the ‘Import Optional Values For Parameters’ button.
       


 1. How to import optional values from Excel file?
 
     1.1.    Select the ‘Source Type’ as Excel.
     1.2.    Click on the Browse button and select the path of the excel file.
     1.3.    Click on the Template button. This will open the file with a sheet by name same as the API model name.                                    The sheet  will have the format in which the values need to be entered.

              


     1.4.    Enter the values in the given format. Eg. as shown below values in row no. 2 are entered manually. Row no. 1 was the heading provided in the template.
              Note:- The parameter name is case sensitive. Hence only the exact matched parameters will be added.

                  

    
     1.5. If Template button was clicked and entries were created in the new sheet then user has to browse the file again to get the latest sheet.
             Select the template Sheet from the drop down 
                

     1.6. ‘Select Row(s) Where’ field is displayed. The user can give any condition if required in this field.

                 

             If user wants to filter a field that has space in its name then use the below format:-
             Name in square brackets.
             [Name]

                 

     1.7. Click on the ‘View All Data’ button to view the content in the sheet. Click on the ‘View Filtered 
            Data’ button to view the data that is filtered after applying the given condition.
            Click on the Next button once parameters are viewed.

                  

     1.8.  The below screen is displayed which will display the list of available parameters that can be added.    
             Select the desired parameters and click on the Finish button.
                   

     1.9. Click on the Ok button of the message box.
              


     1.10. As shown below all the optional values from the Excel file get added to the respective parameter names.
               


2. How to import optional values from Database?

     2.1.    In the ‘Source Type’ select the Database option.
               There are 2 options, Oracle and MSAccess.

                
  
     2.2.    For Oracle Database, enter the connection string, user id and password. Click on the Connect button.
               Free SQL field is populated once the connection is successful.
               Enter the desired query in it and click on Next button.

                 

               Examples of Oracle string:-
              TNS\Host\File Path : 
              Data Source=(DESCRIPTION=(ADDRESS=(PROTOCOL=TCP)(HOST=illin056)(PORT=1521))(CONNECT_DATA=(sid=ATSTSTDB)));User Id=GingerTST;
              Password=GingerTST; 
              TNS\Host\File Path :  DESCRIPTION=(ADDRESS=(PROTOCOL=TCP)(HOST=illin056)(PORT=1521))(CONNECT_DATA=(sid=ATSTSTDB)))


      2.3.    The result of the query is displayed. The fields are displayed and not the values stored in the fields. Click on the Finish button to add the values. 

                

     2.4.    It will display a message giving the number of parameters updated
