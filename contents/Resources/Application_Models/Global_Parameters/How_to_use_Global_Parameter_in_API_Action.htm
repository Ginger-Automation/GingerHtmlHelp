How to use Global Parameter in API Action?

 

1.    For adding a new API Action follow the below steps:-
    a)    In the Automate tab, select an Activity with Target application as ‘MyWebServicesApp’. 
           Click on the Add Action button and select the Web API Model action.
             

    b)    Select the desired API having the global parameter and click on the Select button.
             

    c)    Below screen is displayed giving the API details. Click on the Next button.
             

 

    d)    As shown in the below screen, the global parameter is also displayed. Click on the Finish button.
          
 

    e)    The Action gets added in the grid.
          

2.    When global parameter is added to an API model which is already used in an Action then follow the below steps.
        Refer the “Auto Pilot- Application Models” section on how to create the API Action.

       Open an existing API Action in edit mode by double clicking on it. Below message is displayed. Click on the Ok button.
       

 

       The added global parameter should be displayed in the API Model Parameters list as shown below.
