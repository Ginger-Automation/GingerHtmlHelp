How to Configure Global Parameters?

1.    Introduction
       Global parameters are used in Auto Pilot- Application Model Actions. They are introduced so that parameters can be used across all the        Application Models in a solution.

2.    How to configure Global parameters?
      1)    How to Add global parameter?
            a)    Go to Resources tab and click on the ‘Application Models’. 
                   Then click on the ‘Models Global Parameters’ link as shown below.

                    

            b)    Click on the Add button as shown below

                    

 

            c)    A new window is displayed where user needs to enter the desired global parameter name and click on the Ok button.
