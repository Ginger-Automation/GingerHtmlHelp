Documents

How to use Documents?    
Documents is the area where we can store the documents required by the solution at one place in the Solution folder.
User can edit the documents from the Editor area.
New documents and either be created or imported in this section.

Go to Resources tab and click on Documents.


To open the document in the File folder format, right click and select the ‘Open Folder in File Explorer’ option.  
User can paste the desired documents here.
 


 


Click on the Refresh button from the toolbar and the Documents will be displayed in the Document section as shown below.
