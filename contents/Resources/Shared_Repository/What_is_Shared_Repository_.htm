What is Shared Repository?

1.    What is Shared Repository?    
        A shared repository is a centralized repository within Ginger for sharing Variables/Actions/Activities/Activity Groups across all the Business flows 
        present in the solution. Sharing ensures reusability of the components. In GINGER the smallest unit of reusability is an Action.
        This is Ginger Tool Feature independent of any platform.

2.    How to View Shared Repository?
        i)  Shared Repository is visible in the Business Flows tab. Double clicking on the desired Business Flow.  Click on the 'Add Actions' button. Select Shared Repository.
