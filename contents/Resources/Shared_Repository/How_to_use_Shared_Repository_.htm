How to use Shared Repository?

      The items in the Shared repository can be used in other Business Flows of the same Solution.
   

  1.   Double click on the desired Business Flow in Business Flows tab. Click on the 'Add Actions' button and select 'Shared                                    Repository'.  Select the repository item. Drag and drop it to the area where the item belongs to. 
               Eg. Action can be dropped only in the Action area of the solution.
                


      2.     Another way to Add the item to the solution is by using the arrow button (Add to Flow) as highlighted below.
                   

     3.       Click on the added item. The orange color star icon shows that this item is in Shared Repository.
                 When the color of an item is Grey, then it means that the item is not in shared repository.
