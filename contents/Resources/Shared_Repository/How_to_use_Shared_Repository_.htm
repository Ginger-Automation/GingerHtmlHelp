How to use Shared Repository?

      The items in the Shared repository can be used in other solutions.
       Go to the tab in the Shared repository, select the item. Drag and drop it to the area where the item belongs to. 
               Eg. Action can be dropped only in the Action area of the solution.
                


          •    Another way to Add the item to the solution is by using the arrow button (Add to Flow) as highlighted below.
                   

          •       Open the item in the Solution. The orange color star icon shows that this item is in share repository.
                  When the color of an item is Grey, then it means that the item is not in shared repository.
