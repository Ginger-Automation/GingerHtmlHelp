How to Edit Item in Shared Repository?

1.    The Items in the shared repository can be modified using the pen icon (Edit) as highlighted below.
           


2.    This opens the Edit window as shown below. Make required changes.
            

3.   Click on Save button, a message is populated for confirmation of changes if the item is used in the solution.
         


4.    On clicking the Yes button, the repository gets updated. This does not update its instance in the solution.
        Clicking on the No button does NOT update the repository.
