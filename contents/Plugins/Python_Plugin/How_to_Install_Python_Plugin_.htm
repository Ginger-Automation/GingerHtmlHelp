How to Install Python Plugin?


Purpose
The Python Plugin target is to allow executing Python scripts from Ginger as integrative part of the Ginger test flow.

Ginger Python plugin allows to:
•    Configure python script execution by using script file.
•    Configure python script execution by using script content.

For above execution methods ginger python plugin offers:
•    Send execution Input Values to python script. 
•    Get execution output values by parsing and showing console values and look for “key=value” pairs                 


Pre-requisites
Please see below the items which needs to be set up on the testing environment before executing the Python plugin.
    1.  NET Core SDK Installation
        The plugin requires .NET Core SDK version 2.2 or above to be installed on the machine it is running on.
        Please install it from below link:
        https://dotnet.microsoft.com/download

    2.    Python Installation
        Python installation is must for the plugin to work.
        “PYTHON_HOME” environment variable system configuration is also required.

    3.    Script 
        The python script files need to be located on the machine on which the Python Plugin running.
        It is recommended to place the script under the Ginger Solution “Documents” folder and use relative paths to it (like “~/Documents/Scripts”) so it will work on any user machine. 

How to Install Python Plugin?
Ginger python plugin needs to be installed to the Ginger Solution to be used.
Please follow below steps to perform it.
    
    1.    Go to Resources->Plugins -> Download Plugins
    2.    Select Python and click on the “Install” button              

          

     3. Once installation is complete, Uninstall button will be displayed

           

      4. Go to “Installed” tab and confirm ”Python” plugin is displayed.
