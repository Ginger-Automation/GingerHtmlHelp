How to Execute & get Output Values from Python Script?


1. How to Execute Python Script?
    Please follow below steps to execute python script.

   For this the Python plugin should be installed. For this refer to the topic "How to Install Python Plugin?".


    1. How to execute python script using external file?
        a.   Double click on the desired Business Flow .
        b.   Click on 'Add Actions' button.

                  

        c. Select Actions Library in loaded window go to “Plugins Actions” tab. Select “Run Script file” Action and click on the 'Add to Actions'  button.

                
     


        d.    

                

                Set Action configurations according to the below options:

2.    Get Output Values from Python Flow to Ginger Flow

          Python plugin supports getting output values from the Python Flow.
          It been done by parsing all values which the Python flow write to the Console during execution, the parsing mechanism looks for any            “key=value” pair and if it been found so the Key and Value been added to the Action Output Values list.

       How to Get Output Values from Python Flow to Ginger Flow
          1. Prepare the Python Flow Code for Sharing Output Values
                a.    Open the Script in notepad or another Editor.
                b.    Look for the places in code (either tested class methods or actual test methods) which have values that can be shared with Ginger during execution

           2. Configure Ginger Python Plugin Action to Read Output Values
                a.    Configure any one of the Python Actions configurations
                b.    After execution end, go to Action “Output Values” tab and see all output values which were parsed in run time
                c.    Now you can set Validations on top of those values or pass them to Variables/Data Source or other to be used in next Actions/other in the Ginger flow
