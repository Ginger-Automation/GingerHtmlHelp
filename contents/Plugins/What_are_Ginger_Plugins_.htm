What are Ginger Plugins?

Description    
Plugins are add-ons that adds specific feature to ginger, we are using Plugins to enable third party developers
To create abilities which extend ginger and to easily adding new features.  
We have two Plugins types. 
    •    Embedded plugin – All files are being copied to the solution folder and will be run from there.
    •    System Plugin - All files are on the local machine and plugin will run from local machine.

Pre Requisites
    1.    Latest Ginger version.
    2.    Plugin DLL and all references.
    3.    JOSN file – configuration file .
    
    
1.    How add a new Plugin?
    a.    Right click on plugins and Add new plugin.
            

    b.    Select the folder location of the plugin and click open

    c.    once plugin Folder was selected you will get new plugin Page 
        On this page you will be able to see all the Information about the plugin 
            1.    plugin Name
            2.    Description 
            3.    Plugin Folder – relative path for embedded plugin and full Path for system plugin.
            4.    Plugin type -   embedded/ system.
            5.    Plugin Action – list of all available action on this plugin  
            6.    Plugin Text Editors – if plugin supports text editors , it will be listed on 
                The text Editors tab.
               

      

2.    How to add a Plugin Action?
    a.    In Order to add new plugin Action move to the automate tab and click on 
          Add action button, if you have plugin loaded then you will see new tab on the add Action page, plugin Actions.

             

         

    b.    Select new plugin action and add it to your business flow.
