How to Edit PACT Document?

    Pre-requisites: 
    •    Pact PlugIn is installed.
    •    Pact File Created or Imported to the Documents Folder.

    1.    From the left tree menu expend the “Documents” folder and select the .pact file.  The PACT Content Editor  is displayed

                


    2.    Editor functions:
           a.    Highlighting of saved words (Interaction:, Given, Upon Receiving, Method etc.)
           b.    Comments highlighted in green when the text is after “//” or between “/*” and “*/”
           c.    Tags highlighting             
           d.    Each value will be highlighted in blue when given between quotes
           e.    Table Editor Page will help you add new values to the Headers variable:
                   •    Click on the table you want to edit
                   •    Add/remove values from the below table page
                   •    Click on update


    3.    To Save the updates, right click on the document and click on the Save option.
