How to Edit PACT Document?

    Pre-requisites: 
    •    Pact PlugIn loaded.
    •    Pact File Created or Imported to the Documents Folder.

    1.    From the left tree menu expend the “Documents” folder and select the created or imported .pact file 
    2.    Once the file selected you will see the PACT Content Editor 
    3.    Editor functions:
        a.    Highlighting of saved words (Interaction:, Given, Upon Receiving, Method etc.)
        b.    Comments highlighted in green when the text is after “//” or between “/*” and “*/”
        c.    Tags highlighting
        d.    Folding - Each Interaction can be collapsed and extended by clicking  (-) and (+) buttons 
        e.    Each value will be highlighted in blue while it between quotes
        f.    Table Editor Page will help you add new values to the Headers variable:
            •    Click on the table you want to edit
            •    Add/remove values from the below table page
            •    Click on update
    4.    Save button will save the file imported under the solution
    5.    Undo button will undo the last action
    6.    Delete Selected button will delete the selected section
