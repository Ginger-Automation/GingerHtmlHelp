How to Create Or Import PACT Document?

1.    How to Create Pact Document?
    Pre-requisites: 
    Pact PlugIn should be loaded.
        1.    From the left tree menu right click on “Documents” folder 
        2.    Select “Created Document” -> “Create Pact Document”
        
             

             Note:- if you can’t find “Create Pact Document” option you probably haven’t loaded the plugin as expected

        3.    Enter Pact file name
        4.    Pact file created with text from the pact template, once the new tree item selected you will see the Pact Content Editor

               


2.   How to Import Pact Document?
    Pre-requisites: 
    Pact PlugIn should be loaded.
    1.    From the left tree menu right click on “Documents” folder 
    2.    Select “Import Document” -> “Import Pact Document”

           
           Note:- if you can’t find “Import Pact Document” option you probably haven’t loaded the plugin as expected

    3.    On Import PACT Page browse for file location and then click Import.

          


    4.    The  Pact file imported, once the new tree item selected you will see the Pact Content Editor
