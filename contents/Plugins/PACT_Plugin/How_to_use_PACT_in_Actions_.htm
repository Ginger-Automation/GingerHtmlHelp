How to use PACT in Actions?


    1.    Double click on desired Business Flow under the Business Flows tab
    2.    Click on 'Add Actions' button.
    

           

 

     3. Select Actions Library in loaded window go to “Plugins Actions”.  Select the desired action and click on the "Add to Actions" button. 

            
          

         
 
  1.    Start PACT Server
            Pre-requisites: 
            Pact PlugIn should be installed.

             


            1.     This action should be used for starting the PACT server in order for it to be ready to get Interactions load.
            2.    Configure free port on the port field.
            3.    If Use Dynamic port is checked, Ginegr will search for a free port on your machine and will use it to start the server.
            4.    Run Action
            5.    When Running the Action – the PACT service is started at: http://localhost:1234 with the port specified.
            6.    On Extra Info Field (Can be seen on the actions grid or from the start server action edit page on another tab) you will find the server location it been started. http://localhost: <<port specified>>

  2.    Load Interaction File
            Prerequisites: 
                •    Pact PlugIn installed
                •    Pact File Created or Imported to the Documents Folder
                •    Pact File Compiled
                •    Pact File Exported to JSON
                •    Pact Server Started
                   

            1.    This Action should be used to Load Interaction JSON file in order to fire real API requests against the PACT server.
            2.    Open the Action Edit Page
            3.    Browse for the Exported Json file
            4.    Run Action
                Note: The server should be up and running while executing this action

 3.    Clear Interaction
            Pre-requisites: 
                •    Pact PlugIn loaded
                •    Pact File Created or Imported to the Documents Folder
                •    Pact File Compiled
                •    Pact File Exported to JSON
                •    Pact Server Started
                
            1.    This Action should be used to clean all interaction which been loaded to the pact server
            2.    This action doesn’t required any configuration
            3.    Simply add the action and run it in order to clean all loaded interactions from the server

 4.    Stop PACT Server
            Pre-requisites: 
                •    Pact PlugIn loaded
                •    Pact Server Started
                
            1.    This Action should be used to stop the pact server
            2.    This action doesn’t required any configuration
            3.    Simply add the action and run it in order to stop the Pact server
