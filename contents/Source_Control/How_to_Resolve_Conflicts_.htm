How to Resolve Conflicts?

When there are multiple users working on the same solution at the same time then there are chances that conflict arises during the Check-in process.
Ginger will display message in such cases.

    There are 2 options to resolve these conflicts.
      1)    Keep Local Changes
            In this option, only your changes will be saved in the SVN. The changes made by other users will not be uploaded to the SVN.

              

            

      2)    Accept Server Changes
            In this option, changes made by other users that our updated in the SVN will be downloaded. Changes made by you will be lost.
