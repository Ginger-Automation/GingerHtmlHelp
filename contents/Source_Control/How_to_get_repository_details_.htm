How to get repository details?

In the Solution tab, click on the “Repository Details” button.

                  


 The details about the Solution stored in SVN is displayed. 
 It gives the url of the solution in SVN, the path of the solution in the local PC, Revision number and other details.        

    

 

How to get info of repository of a Ginger component?

     Right click on the component. Go to Source Control and select “Get Info” option.
      

 

    The details about the Component are displayed. 
    It gives the path of the XML File in which the component is stored on the local PC.  
    Revision number from the SVN is also displayed along with the Change Date & Time.
