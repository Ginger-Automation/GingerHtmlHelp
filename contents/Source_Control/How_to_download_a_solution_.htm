How to download a solution?

 

1.  How to download solution from SVN Repository?

        1)    Click on Solution name drop down and select “Download Existing Solution” option.

               


        2)    In the displayed window, enter the login credentials, the SVN URL and click on the “Download Selected Solution” button.
                Note: Ginger uses url :- http://cmitechint1srv.corp.amdocs.com:81/svn/       
                

                               
        3)    Once the login is successful then the “Connect and Search Repositories” button gets enabled.  
               Click on the button.
               List of solutions are displayed for which the user has access to.
               

        4)    The desired folder where you want to store the solution on your local PC can be given in the “Local Solutions Folder Path” using the “Browse” button.
             

 

        5)    When there are multiple solutions displayed in the list, you can search your desired solution using the “Search” button. 

                 

        
        6)    The column “Exist Locally” helps to determine if the solution was already downloaded on your local PC. If yes then it is displayed as “True” else “False” is displayed.

               


        7)    Select the desired solution and click on the “Download Selected Solution” button.
               A message is displayed about the successful download. I also wants to know if you want to open the Downloaded Solution.  
              
    
        8)    Click on Yes to open the solution.

                

 

2.    How to download solution from GIT Repository?

          1)    Click on Solution name drop down and select  “Download Existing Solution” option.
                   

           2)    In the displayed window, select the GIT open in the Source Control type.
                    Enter the login credentials, the GIT URL.

                    Expand the Advanced Connection Configuration and click the “Configure Proxy” check box.
                    Click on the “Download Selected Solution” button.
                    Note: Ginger uses url :- https://github.com  

                        

 

                    

                 
           3)    Once the login is successful then the “Connect and Search Repositories” button gets enabled.  
                    Click on the button.
                    List of solutions are displayed for which the user has access to.
                   


           4)    The desired folder where you want to store the solution on your local PC can be given in the “Local Solutions Folder Path” using the “Browse” button.
                    

          5)    The column “Exist Locally” helps to determine if the solution was already downloaded on your local PC. If yes then it is displayed as “True” else “False” is displayed.

                

           6)    Select the desired solution and click on the “Download Selected Solution” button.
                    A message is displayed about the successful download. I also wants to know if you want to open the Downloaded Solution.  
                   
 
          7)    Click on Yes to open the solution.

3. Copy your solution folders inside this downloaded solution.

              

 

 

        

 

             

 

4. Open the SVN solution in Ginger
