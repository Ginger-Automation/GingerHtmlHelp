How to Execute Automated Flows Which Were Designed in Design Console?

1)    Export you Designed Console designed flows to ALM as Test Sets
2)    Import the Test Sets from ALM into Ginger as Business Flows – all automated Test Cases will be pulled 
              from Ginger repository automatically.
3)    Execute the Business Flow in Ginger

      Note: Please review the Ginger-ALM (QC) Integration user guide for more detailed information
