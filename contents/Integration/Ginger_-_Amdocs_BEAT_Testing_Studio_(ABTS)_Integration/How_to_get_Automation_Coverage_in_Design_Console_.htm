How to get Automation Coverage in Design Console?

 Introduction    
    •    Projects which use ALM (QC) and BTS for their manual testing design are able to automate their ALM Activity Library using Ginger.
    •    The automation coverage for their designed flows is visible in the Design Console of the BTS in real time.
    •    After exporting the scenarios to ALM from BTS, the users will be able to import the ALM Test Set’s to Ginger and execute them according to the design in BTS.

How to get Automation Coverage in Design Console?
    1.    Export your Design Console designed flows to ALM as Test Set’s
    2.    Import the Test Set’s from ALM into Ginger as a Business Flow
    3.    Automate the Test Set’s in Ginger
    4.    Add the automated TC’s to Ginger Shared Repository
    5.    Upload the Ginger Shared Repository changes to the Ginger Source Control (SVN)
    6.    Click on “Get Automation Coverage” button in Design Console
