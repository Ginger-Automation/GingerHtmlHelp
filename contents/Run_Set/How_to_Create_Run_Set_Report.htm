       Note: In case the Run set was not executed as a full execution or in case Stop-Continue was used then 
                HTML reports won’t include all iterations of executed items.

3.    Execution History tab

      Click on the Execution History tab. After the Run set is executed, initially the history is displayed as blank, 
      click on the Refresh button to get the history populated.
      ‘Generate Report’ button will populate the HTML report and open the folder where all the reports are stored.
