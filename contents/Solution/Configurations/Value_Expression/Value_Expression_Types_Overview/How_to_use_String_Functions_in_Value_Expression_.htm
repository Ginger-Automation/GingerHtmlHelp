b)    Trim whitespace  & Line breaks– It discards the whitespace and line breaks that occur in the given string.                                                       String can be either a variable or a text field copied from some page.

                   


c)    Get Inner String Index – it gives the number at which a given string begins in the main string.                     

       


d)    Replace Text

      The content “abc” needs to be replaced with either a variable or some text to get the desired result.
          


e)    Text Contains
       The content “abc” needs to be replaced with either a variable or some text to get the desired result.
          

f)    Replace Text
       The content “Abc” needs to be replaced with either a variable or some text to get the desired result.
       “A” needs to be replaced with your desired text.
          

g)    Ends With

       The content “Abc” needs to be replaced with either a variable or some text to get the desired result.
       “c” needs to be replaced with your desired text.

         

h)    Get String Length

       The content “Abc” needs to be replaced with either a variable or some text to get the desired result.

        

i)    Get Substring

      The content “Abcdefg” needs to be replaced with either a variable or some text to get the desired result.
      Also the numbers 1 & 3 can be replaced with the number of your desire.
