How to Add Value Expression?

   
    •    The Value Expression can be added for an action using the ‘…’ button in the Edit Action window as shown below.

          
       
    •    The below screen is displayed. It displays the variables (Global, Business Flow, Activities and Environments). Ginger provides the functions from VB Script, Regular expressions and Data Source expressions.
