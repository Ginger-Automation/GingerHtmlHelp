Target Application

A Target application is an application on which the execution is going to be done.  

A business flow can have multiple target applications.

An activity can choose one target application form the target application configured in Business Flow.


Supported Platforms for Target Applications

  1. Android devices – used when the testing application is android based application.

  2. DOS – used for DOS based applications

  3. Unix – used for unix based applications

  4. Java - used for Java based applications liked CRM

  5. MainFrame  - used for main frame based applications

  6. Mobile - used for Mobile applications

  7. Power builder - used for power builder applications

  8. VB Scripts - used for Visual Studio applications

  9. Web - used for Web applications

  10. Web Services - used for web applications related to services

  11. Windows - used for Windows applications
