Agent Configuration

Go to the Configurations tab and click on Agents. Click on the + button to add new Agent.

                 


  The below wizard is displayed. It gives information about the Agent and its usage.   
  Click on the Next button to add a new agent.

       

 

   Select the desired Agent. Enter the Name, description, select the Platform type, Driver type and click on the Next button.

       

 

The Driver Configurations is displayed where the details can be entered and click on Finish button.
