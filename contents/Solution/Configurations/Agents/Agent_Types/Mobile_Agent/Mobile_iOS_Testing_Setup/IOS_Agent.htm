IOS Agent

i)    Open the Add New Agent Wizard and click on the Next button.
      Enter the name, select the platform as Mobile.  Driver as Mobile Driver and Types as MobileAppiumIOS.

       Click on Add button to associate a Tag if required. Click on the Next button.
      


ii)    Enter the AppiumServerIP & AppiumServerPort.

       Click on ‘Test Agent’ button. Connection successful message is display if connection is a success.
       Driver load time configuration is displayed and can be updated if required.
       Click on the Finish button to add the agent.
