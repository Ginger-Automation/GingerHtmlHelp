Mobile iOS Troubleshooting

    1.     Issue - Failed to Connect to Device
              Possible Solution:-
                   Make sure the tested device is not locked

 

    2.    Issue - Failed to load Safari Browser
              Possible Solution:-
               1.    Especially on iPad- Make sure that Safari is been launched on the device only with only one tab.

                      If other tabs exists then:
                       o    Close all tabs by clicking on the tabs control button in Safari browser and closing all existing tabs before doing the Ginger-Appium connection
                       o    Clear all browsing History data from Safari app or Settings before doing the Ginger-Appium connection
                2.    Make sure ios-web-proxy is not crashing- if it is, check for the crash reason:
                       o    If ios-web-proxy saying it failed because “Unknown app ID” then uninstall the SafariLauncher app from the device and re-run (it will re-installed by appium)
