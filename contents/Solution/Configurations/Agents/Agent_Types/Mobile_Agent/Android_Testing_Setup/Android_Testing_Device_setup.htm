Android Testing Device setup

1.    Setting Device Configurations

        a.    Set the phone to “USB Debug Mode”:
             •    Enter the phone System Settings-> General-> Developer options
             •    Turn on the USB Debugging option
                 Note: in case you can’t find the “Developer options” then go to System Settings-> General-> About phone-> Software

                             information-> Click 7 time on the “Build number” then the Developer options supposed to be added
        b.    Connect the phone to the PC using USB
             •    The correct phone driver should be automatically be installed. 
                     If it isn’t:
                       o    For window: Install PDANet from http://pdanet.co/bin/PdaNetA4150.exe- as part of the installation it will allow

                              you to install a driver which fits to your phone.
                              You can also search for relevant driver online.
                       o    For Mac- search online for the relevant driver for your device
        c.    If you been asked on the device to approve the PC – then do it


 2.    Getting Device Details 
 
          a.    Connect the device to the PC using USB 
          b.    Get the Device ID
                   o    On Windows:
                        i.    First Method:
                             1.    Open CMD
                             2.    Type “adb devices”
                             3.    Get the ID of the device from returned list of connected devices
                                            

                                      

                        ii.    Second Method:
                             1.    Open the Android SDK monitor tool by doing StartRun “%ANDROID_HOME%\tools\monitor.bat”
                             2.    You supposed to see the connected devices in the top left list
                             3.    Part of the name is the device ID                                    

                                      
                                         

                   o    For Mac:
                        i.    Open Terminal window
                        ii.    Type the command below and type enter
                               adb devices
                        iii.    List of all connected android device supposed to show with their device ID
                                

                                
                               Note: if the list is empty then make sure that the device driver is installed and that it is connected

                                          (also try to reboot your Mac machine)
