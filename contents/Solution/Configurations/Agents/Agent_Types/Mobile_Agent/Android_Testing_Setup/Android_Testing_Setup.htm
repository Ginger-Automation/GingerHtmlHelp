Android Testing Setup on Appium Server

These steps will allows Ginger to run tests on an USB connected Android device using Appium.

 

 Pre-requisites
    i)    Ginger need to be installed on a Windows operating system machine.

    ii)    In server can be installed on:
           Windows OS that supports .NET Framework 4.5.1
                   1.    Windows Vista SP2
                   2.    Windows 7 SP1
                   3.    Windows Server 2008 SP2
                   4.    Windows Server R2 SP1
                   5.    Windows Server 2012


   iii)    Ginger Windows machine and Appium server machine must be located on the same network (can ping each other)
   iv)    Admin user to the Ginger Windows’s machine and Appium server machine is required for installations
 
Ginger- Appium Connection Diagram

            


Appium Server Installation
        All installation steps need to be done on the Appium sever machine using an Admin user.

        1.    On Windows Machine
             1.1    Windows: Install Appium Prerequisites for Windows
                   1.1.1  Install latest Java JDK:
                            a.    Download the latest Java JDK version from below link (which fits to OS type 32/64 bit):  
                                   http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html
                                   


                            b.    Double click and install it- Make sure Java JDK is saved in a folder that is accessible to all users (e.g. “C:\Program                                                  Files\Java\jdk1.8.0_25”)

 

                   1.1.2  Install Android SDK:
                           a.    Download the latest Android Studio (SDK is included) from http://developer.android.com/sdk/installing/index.html
                                  
                           b.    Run it - Make sure Android SDK is saved in a folder that is accessible to all users (e.g.  “C:\Android\Android SDK”)


                   1.1.3    Set the Environment Variables:
                          a.    Go to Control Panel-> System and Security-> System->Advanced system settings
                                  


                          b.    Go to Advanced tab->Environment Variables…
                                 


                          c.    Add/Edit the below System variables:
                                 i.    Variable Name: JAVA_HOME
                                       Variable Value: [path to Java SDK] (e.g., “C:\Program Files\Java\jdk1.8.0_25”]
                                       


                               ii.    Variable Name: ANDROID_HOME
                                      Variable Value: [path to Android SDK] (e.g., “C:\Android\Android SDK”]

                               iii. Variable Name: Path
                                    Variable Value: %PATH%;%JAVA_HOME%\bin;%ANDROID_HOME%\tools;%ANDROID_HOME%\platform-tools;
                                    Note: If Path variable is already exist then just add the above value to the existing value


            1.2    Windows: Install Appium 
                     a.    Open the Appium Web site:
                             http://appium.io/
                    b.    Click on Download Appium button
                            

                    c.    Double click and install Appium application- install it in a folder which will be accessible to all users

                    d.    Launch the application from Start->Appium->Appium
                            

                              

                               


          1.3    Windows: Configure and Launch Appium Server
                     a.    In case you plan to run Ginger which installed on a different machine (same network):
                           i.    Open windows Run type “CMD” and click enter in the opened window type “Ipconfig”
                          ii.    Write the IP value which appears for “IPv4 Address”
                                       


                        iii.    In Appium, click on “General Settings” button and replace the “127.0.0.1” value with the IP number you got in

                                  the previous step 
                                            

                        iv.    Close the General Settings window
                        v.    Click on “Launch the Appium Node Server” (Play icon)  

                                     

                       vi.    Make sure appium is starting without errors
                      vii.    Appium server is UP and ready to get connecting request from Ginger!
