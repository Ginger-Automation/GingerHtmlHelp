Perfecto User guide

   Perfecto cloud supports mobile testing while connecting to perfecto cloud on real devices


1.    Agent Configuration


    1.1  Add New Agent
 
           Give it a name and select Mobile as the agent type then select one of the 4 Perfecto options according to the Device type,                               Android or IOS device and Native (regular App) or Web Browser testing.’
            
           

             

 

            On Agent configurations populate the Cloud URL, User Name, Password and the device ID

            

            
    1.2  Populate Agent details
                
            On Agent configurations populate the Cloud URL, User Name, Password and the device ID

              

 

            To get the device id go to perfecto cloud click lunch device and take the id from the grid:

                 

              

 

    1.3  Start Agent

            Go to BusinessFlow page and start the Agent – agent should become Green, if the connection will fail the Agent will remain red                    and the connection error will be populated in the log.

              

 

2.    Automation & Execution

    2.1 Dashboard
           Perecto supplies a dashboard for the connected device, Ginger will open the browser while starting the agent, in order to see the                     device click on select device and select the device from the grid.
                

               


    2.2 Action Types

           We have 2 types of mobile actions.
            •    Mobile Device action
            •      UI Element Action

           2.2.1 Mobile Device Action

                     To perform actions on the device level like swipe, type, volume buttons etc. use the Mobile device action.

                       Click on the 'Add to Actions' button to add the action. 

                         

 

                       Double click on the added Action to open it in Edit mode.

                         

                           

                          

 

           2.2.2 UI Element action

                    Select the UIElement action as shown below and click on the 'Add to Actions' button.

                                 

                             

                           To get more details on how to use Locate by & Locate Value, refer to topic 'Locate by -Locate Value'

                                   

 

                        In order to grab values of locators (id xpath name etc) go to perfecto site, click on lunch mobile device, click on
