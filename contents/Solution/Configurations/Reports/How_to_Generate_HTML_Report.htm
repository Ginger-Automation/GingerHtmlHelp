How to Generate HTML Report

1.    Runset Configuration on the Run tab

 

        i)      Go to “Run” tab, select your “Runset”.
                Click on “Configuration” tab.

                      

                Here user can update the name of the Runset, add description and select if Analyzer is required before manual 
                execution of the runset.

 

        ii)    Producing Reports of the execution

               Here user can add various activities that will be done before/after  runset execution.
               The one of a such activities is the “Add Producing HTML Report Action”.
                Go to the Operations tab and click on relevant button. (there is no limitation on quantity of added actions (meaning - produced reports) per one execution here.
                

               By adding this action to tray -> one report will be created at by the finish of execution.
               Used template of report can be changed here (default selection is the “Default” report)
               Also folder for saving current report can be changed here (to be deferent than one that considered at bullet 2, second figure)

                 


               Once such action added – report will be created per each execution, with options that described in this action.
               Folder with new created report will be opened automatically by windows explorer.
    

      iii)    Sending Reports via email

              User can send reports via email before/after  runset execution.

              Go to the Operations tab and click on ‘Add Send HTML Report Email Action’. 

               

 

             Enter the desired email ids in the From and To fields, subject and the body of the email.

                 


             Go to the attachments tab and click on ‘Add Report’ button. This is will send the HTML Reports as attachment in the mail.
             Run button is displayed at the bottom to check if the test mail is delivered successfully. 

                   

 

             Click on the Save button to Save the updates to the Runset.

                 


2.    Run set list


             Presentation of all RunSet executions (that are done at “Run” tab) is available as list.
             Reports per execution also can be produced – it can be done instantly and on demand. 
            For customization of such report will automatically be used a default template.
            In order to see this presentation -  left click on “Execution Results” – “Execution Results Configuration” (“Solution tab”)

                 

 

            “Search” RunSet execution, “Configure” execution creation of results, “Refresh execution results”, “Delete” and “Open selected  execution results folder” functionality is available at this window.
             By clicking on icon at “Generate Report” column at specific row – generation of report is done. 

                   

    

             Folder with new-created report will be opened automatically.

                   

 

3.    Generate Last Executed Report

 

            It is possible to create the report for the last execution that done via “Automation” tab.

            For this - run one from 2 possible ways to do the execution – “Run Action”, “Run Activity” or “Run Flow”.
            Once it’s done - you able to process the report – click on “Generate Report” -> “Generate Last Executed Item Report”   or “Generate Full Business Flow Report”
