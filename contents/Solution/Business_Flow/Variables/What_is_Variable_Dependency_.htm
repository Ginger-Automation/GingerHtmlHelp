What is Activity Variable Dependency?

     1.    To Activate Variable Dependency, Click on the Activities-Variable Dependencies.
            

 

     2.    In the pop window, enable checkbox Activities-Variable Dependencies Control. Click OK
             

 

     3.    Then click on Business Flow Variable and add one selection list variable.
             

    4.    Open Edit page of variable selection list and add optional values in that list.
            

    5.     Open Activities-Variables Dependencies page as shown in Step 1. Now we can see the new extra columns of variable selection list values allowed by activities.
            

    6.     Here for variable selection list: a, selected activities are 1st and 2nd that means for variable value a, only those activities gets executed. 
           Similarly, for variable selection list: b, selected activities are 1st and 3rd that means for variable value b, only those activities gets executed.
           

    7.    By default variable selection list value is first means here selected value: a , so only 1st and 2nd activities gets executed and remaining are marked as skipped.
           Let’s move towards the execution, click on Run Flow you can see execution like below.
             

   8.    Now open the edit page of variable list and change the selected value: b.
              

  9.    Click on Run Flow and you can see only 1st and last activities gets executed for value b.
