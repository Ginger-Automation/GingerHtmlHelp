How to use Automate Mainframe 3270 in Ginger

1.    Configurations

    a.    Add a new Target Application  with platform Mainframe
          

    b.    Add a New Agent of Type Mainframe 3270
           

    c.    Configure the agent with your application details, Do not change Term Type unless necessary. Enter the HostName and click on the Finish button.
                            


    d.    Add the Target Application to the desired Business Flow
    e.    Add a new activity and choose the Target application with Platform Mainframe.
    f.    On Proper connection you’ll see your application screen with message that the connection is successful.

                              


2.    Add Action
    a.    Set Text To Main frame Action
        Locate By : Use  By Caret Position or BY XY
        ByCaretPosition :  Use the caret position shown in the Ginger mainframe window for the desired location.
        ByXY : We can also use the XY in the format x,y to set the location of the text
        Set Text and Send Enter Checkbox: Tick this if you want to send enter after setting text.

        

    b.    Get Details from MainFrame Action

        Similar to Set Text , Get Details also requires the Locate by and Locate Value .

        The value field is the length of the required text starting from the provided Caret Position/ XY.
        If left blank it will return the text till next whitespace.  
          


   c.    Send Keys to Mainframe
          Use this Action to send Special Keys to mainframe like Enter, Tab Insert, Function Keys etc.
          User just need to select the desired key from the dropdown, no other configuration is required.
