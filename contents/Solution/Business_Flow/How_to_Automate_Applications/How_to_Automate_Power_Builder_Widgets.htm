How to Automate Power Builder Widgets

Description    
Ginger Power Builder Widgets means web page or web browser embedded in power Builder window. 
Widget window contains different elements button, text fields, labels, check box, drop down, radio button, combo box, links etc.

Prerequisite
Ginger and target Power Builder application should be launched from the same machine.

Steps to use Ginger for Widget Operation

    1.    Select Widget Window on Window Explorer Dropdown
           Please follow below steps for selecting widget window:
              1.    Open Ginger
              2.    Start Power Builder CSM agent from Automate tab->Agent
              3.    Open Window Explorer from Automate Tab->Agent->Window Explorer (next to Agent on-Off button)
              4.    Click on Refresh button present on top right of window explorer
              5.    Select Widget Window from dropdown present on top of window explorer

                      

    2.    Find and Initialize Browser element from Tree
           Follow below steps to find and Initialize Browser Element:
              1.    Add Switch Window Action (Mandatory Action )
              2.    Expand tree elements until find the Browser Element 
              3.    Select / Click on Browser Element present on tree
              4.    Run “Initialize Browser” Action present under Element Details & Action -> Action 
              5.    Add “Initialize Browser” Action in solution.

                         

                                
    3.    Find Html Elements on Window Explorer
           Follow below steps to find Html Elements

                3.1.    Find Element by Live Spy:
                          Follow below steps to find element by live spy
                            1.    Initialize Browser element by mention above
                            2.    Click on “Live Spy” button present on top left of window explorer
                            3.    Move cursor on expected element on widget window
                            4.    Press “Control” key of keyboard

                          Expected widget element will get highlighted by red color of border and you will find the expected element details like title,                            value, XPath, available action with different locators on window explorer.

                                    


                3.2.    Find Element on Tree View of Window Explorer:
                          Expand the Tree elements until get the expected Html Element on Tree of window explorer.

                           

                3.3.    Find Element on Grid View of Window Explorer:
                           1.    Click on Grid View Button present on right side on Window element tab on Window Explorer
                           2.    Click on “Refresh Controls” button present on Window element tab on Window Explorer
                           3.    Select expected Html Element from grid.
                                  

                3.4    Perform & Add different Actions on Html Element
                         Follow below steps perform & add action on html elements

                             1.    Find Available Actions & locators in Action tab
                                       a.    After selecting expected html element from Window Element tab (from tree view or grid view), go to Action                                                    tab present under Element Details & Actions tab.
                                       b.    Select different Actions from Actions Tab under Element Details & Action tab
                                       c.    Select different Locators from Locators Tab under Element Details & Action tab

                                               
    
                             2.    Run the Action
                                   You can run the Action by click on Run Action button present on bottom left under Element Details & Action tab of window explorer.
                             3.    Add Action to Business Flow
                                   You can add the Action by click on Add Action to Business Flow button present on bottom left under Element Details & Action tab of window explorer.

                             4.    Find general details and properties of Html elements (Optional)
                                      a.     You can find general details like Title, value, XPath, Type of html element by click on General Tab present                                                         under Element Details & Actions Tab.
                                     b.    You can find different Properties of html element by click on Properties Tab present under Element Details & Actions Tab.
