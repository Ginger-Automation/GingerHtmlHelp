4.    Action drop down:-

    a)    Go to Action:- this will enable another drop down beside it to select the list of Actions from the solution.

              

 

    b)    Go to Activity :- this will enable another drop down beside it to select the list of Activities from the solution.
            


    c)    Go to Activity by Name: in this user can type the name of the activity which needs to be executed.

    d)    Go to Next Action:- will take the flow to the next action.

    e)    Go to Next Activity:- will take the flow to the next activity.

    f)    Rerun Action:- executes the Action again.

    g)    Run shared Repository Action:- this executes the action from shared repository. Action name is given in the field as shown below.
           

 

    h)    Set Failure to be Auto- Opened Defect :- if the action fails, then Ginger will automatically open defect in ALM. For the ALM Connection Details and the Defect Profile must be created.

            

 

    i)    Set Variable value:- to set a value of a variable as shown below

            

 

    j)    Show message box :- when the given condition will be met, then we can direct Ginger to display a pop up message during the execution.

    k)    Stop Business Flow:- this will stop the complete execution of the Business flow.

    l)    Stop Run :- this will stop the currently executed Action/Activity.
