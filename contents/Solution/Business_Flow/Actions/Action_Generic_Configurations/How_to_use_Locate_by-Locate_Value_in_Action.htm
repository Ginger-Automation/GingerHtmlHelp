How to use Locate by-Locate Value in Action Or use Live Spy?

There are 2 ways to do it:-

Populating values Automatically using the Live Spy

      1.1) Launch the agent, in this case website google.

             Click on ‘Add New’ button and click on Live Spy button.

                     

        1.2) Click on the Spy button

                      

 

          1.3) The Spying starts

                         

            1.4) Go to the agent. Hover the mouse to the element you want and press the Cntrl button. When Ginger detects the element, it gets highlighted with pink colour.

                          

 

            1.5) Go back to Ginger and stop the Spy button. The details of the element are populated in this section.

                          

            1.6) Select the desired Locator and click on the 'Add Selected Action to Business Flow' button.

                          

             1.7) The appropriate Action type is added in the Actions list where the 'Locate By-Locate Value' along with other details are automatically populated.
