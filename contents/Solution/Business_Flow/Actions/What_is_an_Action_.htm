What is an Action?

An Action is an entity, in the Automate tab, where executable instructions are given on the desired applications.
The expected results are given here.


This is then compared to the actual results at run time which is used to mark the Action as fail or pass as per the results.

       New Action can be added as shown in the below screen.
           


The Add Action window gives a variety of Actions can be added.

    -    Platform Actions are the ones which are specific to the selected platform based on the Target application eg. Web
    -    Legacy Actions are the actions from older version which have been modified in the latest version.
    -    General Actions are non-platform specific and can be used in any solution.
