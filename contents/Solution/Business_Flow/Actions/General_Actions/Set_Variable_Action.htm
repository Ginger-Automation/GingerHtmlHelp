Set Variable Action

1.    Introduction    
        Set Variables is an Action which is used to create variables in the solution.
       The below are the variable types that can be used in this action:-
          1)    Selection List       
          2)    String        
         
2.    How to configure this Action
       1)    Create a variable (Refer topic "Variables" for more details)

              Here we have created a Business Variable. Click on the Business flow under Buisness Flow tab.                                                                        Go to its Variables tab and click on 'Add New Variable' button as shown below.
                 

       
      2)    The ‘Add Variable’ window should be displayed. Select the desired variable type and 
             click on ‘Add Variable’ button.

              
  
      3)    Double click on the added Variable to open it in Edit mode.  
              Enter the Name & description if required.
            ‘Set as Input Value’ is checked by default.


             Linked Variable can be used when another variable is already created.
             Few variables can have initial value that can be entered in ‘Initial String Value’ test field which gets displayed as the ‘Current value’         

              
   


     4)  Double click on the Business flow under Buisness Flow tab.  Click on ‘Add New’ button and click on Actions Library. 

                                


 
     6)    Go to the ‘General Actions’ tab and select the ‘Set Variable Action’ then click on 'Add to Actions' button shown as below.

                        
 
     7)     Double click on the Action to open it in Edit mode.. Below screen is displayed. 
             Select the variable from the ‘Variable Name’ drop down.
             ‘Variable Type’ is displayed automatically.
              ‘Set Variable Value Operation Type’ drop down displays the action that is required to be done on the Variable.                                                   The options are ‘Clear Special Characters’, ‘SetValue’  and ‘Reset Value’.

               Enter the desired value in the 'Value' field.

                     
 
    8)    Run the Action by clicking on the Run Action button.

            Go the Variable list. The set value is displayed near the variable name.
