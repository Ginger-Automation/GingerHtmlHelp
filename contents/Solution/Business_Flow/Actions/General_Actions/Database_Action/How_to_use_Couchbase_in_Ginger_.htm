How to use Couchbase in Ginger?

1. What is Couchbase? 

   Couchbase DB is an open source database developed by Apache software foundation. 
   The focus is on the ease of use, embracing the web. It is a NoSQL document store database.

2. What does Ginger support in Couchbase? 

   a) Used to connect Couchbase DB and perform operations
   b) Used to connect buckets of Couchbase DB

3. Which Couchbase operations are supported in Ginger? 

   a) Read Data
   b) Write Data
   c) Update Data
   d) Record Count

4. How to configure Couchbase in Ginger? 

   a. Under the Resources tab, go to Environment and select the desired environment. 
   b. Using the + button add the Couchbase DB. Enter name, select DB Type as “Couchbase”, 
      enter the host and user name and password if any.

     

  c. Click on the Test connection button to get the connection successful message.

      
 

5. How to add Database Action for Couchbase?   

    a)    Double click on the Business flow under Buisness Flow tab.  Click on ‘Add New’ button and click on Actions Library.
               
              

    b)    Go to the ‘General Actions’ tab and select the ‘DataBase Action’ then click on ‘Add to Actions’ button shown as below.
              

 


  c) Double click on the Action to open it in Edit mode. Select the environment and the DB and the operation type.
      The DB operation type as same as given in the '4. Action Operation' section of 'How to Configure Database Action?' doc.

        

  d) Run the Action to execute it
