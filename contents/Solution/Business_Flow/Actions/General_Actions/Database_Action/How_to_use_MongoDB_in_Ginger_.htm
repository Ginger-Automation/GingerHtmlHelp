How to use MongoDB in Ginger?

1. What is MongoDB? 

  MongoDB is an open source document database and leading NoSQL database. MongoDB is a cross-platform, document-oriented database    that provides, high performance, high availability, and easy scalability. MongoDB works on concept of collection and document.

2. What does Ginger support in MongoDB? 

   a) Used to connect MongoDB databases and perform operations
   b) Used to perform operations like insert, update, select

3. Which MongoDB operations are supported in Ginger? 

    a) Read Data
    b) Write Data
    c) Update Data
    d) Record Count

4. How to configure MongoDB in Ginger? 

    a)    Under the Resources tab, go to Environment and select the desired environment. 
    b)    Using the + button add the MongoDB. Enter name, select DB Type as “MongoDB”, enter the Connection String or Host and user                    name and password if any.

        Use following ways to connect with MongoDB

        Option 1: "mongodb://user1:password1@localhost/DB" (Connection String format)

        Option 2: "mongodb://HostOrIP:27017/DBName" (TNS/Host/FilePath format)

         

    d)    Click on the Test connection button to get the connection successful message.

         

5. How to add Database Action for MongoDB? 

    a)    Double click on the Business flow. In the Activity Actions, click on ‘Add New’ button

            


    b)    Go to the ‘General Actions’ tab and select the ‘DataBase Action’ then click on ‘Add Action’ button given in the below window

           
 
    c)    Select the environment and the DB and the operation type. The DB operation type as same as given in the                       
            '4. Action Operation' section of 'How to Configure Database Action?' doc.

            

    d)    Run the Action to execute it
