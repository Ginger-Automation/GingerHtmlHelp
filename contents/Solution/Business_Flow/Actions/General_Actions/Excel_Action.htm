Excel Action

1.    Introduction    
       Excel Action is used to read/write data in Excel file. The data in the excel can be used during the creation of customer where lot of 
      details are entered in the form like the name, address, DOB, etc.
2.    How to Configure Action
      2.1 Pre-requisite 
          Excel file is created having Columns and data in it. 
          The column name should be given in braces eg. [customer type]
          Sample file
                           

      2.2    Action Operation

            2.2.1    Read Excel

                       1)  Double click on the Business flow under Buisness Flow tab.  Click on ‘Add New’ button and click on Actions Library. 

                                

                         

                      2)  Go to the ‘General Actions’ tab and select the ‘Excel Action’ then click on 'Add to Actions' button shown as below.

                               

 
                               

                    3)    Double click on the Action to open it in Edit mode. Below screen should be displayed.  ‘Read Data’ action is displayed by default

                            


                    3)    Select the file using the Browse button. From the ‘Sheet Name’ drop down, select the desired sheet.
                           Clicking on ‘View All Data’ button will display the contents in the sheet.

                             

                           If user wants to use a filter whose name has space in it, then use the below format:-

                           The name should be in square brackets:-
                           [Paramter Name]=’<ID>’

                              

                    4)    Enter condition in ‘Select Row(s) Where:’ field Eg. param11=10 or param12=12
                          ‘Primary Key column:’ should be given.
                           Open the Value Expression Editor  ( button next to Select row(s) Where ) as shown to frame a needed condition using the 
                           various expression components.
                           Once done click on OK.

                            


                           Now Click on the ‘View Filtered Data’ button will give the result obtained from the ‘Select Row(s) Where:’ condition.

                           


                    5)    Run Action will read the content as per the given condition and the values will be populated in the Output Values tab.

                            


             2.2.2    Write Excel

                    1)    In order to create a Write action on an excel, it is first required to Read the Excel. Please follow the Read Excel 
                            steps given above.

                    2)    Select the Excel Action as ‘Write Data’ from the dropdown.

                            

                           The ‘Data to Write:’ is used to write the data. Multiple columns can be written using “,”separator.

                           If the excel column’s name has space in it then use square brackets Eg. [customer name]=’Daniel’ .

                           

  
              
                    3)    The Run Action will write the data in the excel which can also be viewed using the ‘View Filtered Data’ button
