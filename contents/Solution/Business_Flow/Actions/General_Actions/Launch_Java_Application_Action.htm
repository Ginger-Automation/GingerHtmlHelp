Launch Java Application Action

  How to Configure Action                      
   

     1)  Double click on the Business flow under Buisness Flow tab.  Click on ‘Add New’ button and click on Actions Library. 

                                

                         

        2)  Go to the ‘General Actions’ tab and select the ‘Launch Java Application Action’ then click on 'Add to Actions' button shown as below.

                             

    3)    Double click on the Action to open it in Edit mode. Below screen should be displayed.  
             •    Option for using the Java Home environment Variable is displayed. The location can be specified.
             •    Java application path can be specified.
             •    Wait option is also available
                     

             •    Java Agent can also be provided         

              •    Java Agent needs to be attached to Ginger in order to do execution.
