Image Search and Click on screen

1.    Introduction    
         Image Search and Click on screen Action is used to search an  element and take mouse action on it.


2.    How to Configure Action
     1)    Go to Automate tab. In the Activity Actions, click on ‘Add New’ button
               

 
     2)    Go to the ‘General Actions’ tab and select the ‘Image search and click on screen’ then click on Add Action button given in the below window
             

     3)    Below screen should be displayed. 
             •    ‘Locate By:’ drop down can be used to indicate the locator.
                   The ‘Locate Value’ can be used using the ‘Open Window Explorer to select Element’ button.
                   Refer the ‘Locate by-Locate Value’ document to know how to use this.

             •    ‘Value’ is entered if required either manually or using a variable.
             •    Main Window title can be given of the element if there are multiple windows open in the Agent
             •    Loca1tor’s Image can be given using the Capture Locator Image button.
             •    Action Type are:-
                   i)    Input Value
                  ii)    MouseLeftClick
                 iii)    MouseDoubleClick
                  iv)    MouseRightClick
             •    Click on Ok to add the action to the Activity Actions.

                  

     4)    On Open of Edit Window 
               a.    Select Action Type
               b.    And click button “Capture Locator Image”.

                   

     5)    Go to the page you want to capture object and select the area as shown below.
     6)    Once selected click on object you want to be clicked while execution. Example Radio button in below image.

                   

     7)    Once capture done you will see following details auto populated on your Edit window.

             
              
     8)    On click of Run Action, it will try to locate the radio button you captured and click.

             
                
             Please Note: Execution might take few seconds to make it locate your object.
