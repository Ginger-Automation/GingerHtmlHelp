Text File Operations

How to Configure Action                      
  

   1)  Double click on the Business flow under Buisness Flow tab.  Click on ‘Add New’ button and click on Actions Library. 

                                


 
     2)    Go to the ‘General Actions’ tab and select the ‘Text File Operations’ then click on 'Add to Actions' button shown as below.

                           
             

                    
    3)    Double click on the Action to open it in Edit mode.  Below screen should be displayed.  
           Select the ‘OperationType’. Default option is Read. Other options are Write and Append.
          ‘Encoding Type’ is available with default option as ‘UTF8’ and other options as ASCII, BigEndianUnicode, Unicode, UTF32 and UTF7.
          Select the ‘Text File Path’ using the Browse button.

          Click on the Run Action button to execute the action.         

                  

 

   4) The output is displayed under the Output Values tab.
