File Processing Action

How to Configure Action
          
    1)    Go to Automate tab. In the Activity Actions, click on ‘Add New’ button
               

 
    2)    Go to the ‘General Actions’ tab and select the ‘XML Processing Action’ then click on Add Action button given in the below window
               


    3)    Below screen should be displayed.
            •    Template File can be given using the Browse button. Files like XML, batch file, etc can be used.
            •    Target File name and Processed File name is mandatory to be given along with their path.
            •    Click on Ok button to add the action.
                                   
                 
    4)    The action gets added in Activity Action list

                
   
    5)    The Run Action button will process the given set of data from the target file into the Processed File as per the  Template given.
