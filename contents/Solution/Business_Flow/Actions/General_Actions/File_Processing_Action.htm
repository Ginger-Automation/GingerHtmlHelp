File Processing Action

How to Configure Action
          
   1)  Double click on the Business flow under Buisness Flow tab.  Click on ‘Add New’ button and click on Actions Library. 

                                

                         

      2)  Go to the ‘General Actions’ tab and select the ‘File Processing Action’ then click on 'Add to Actions' button shown as below.

                     


    3)    Double click on the Action to open it in Edit mode. Below screen should be displayed.
            •    Template File can be given using the Browse button. Files like XML, batch file, etc can be used.
            •    Target File name and Processed File name is mandatory to be given along with their path.

                                                           
   
    4)    The Run Action button will process the given set of data from the target file into the Processed File as per the  Template given.
