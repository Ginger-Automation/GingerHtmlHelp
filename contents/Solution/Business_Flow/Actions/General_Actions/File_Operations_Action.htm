File Operations Action

1.  Introduction    
     File Operations Action is used to various operations on a file. 
     Below is the available list of operations:-
        1)    Check File exists
        2)    Check folder Exists
        3)    Copy
        4)    Execute
        5)    Force Copy
        6)    Move
        7)    Run command
        
2.    How to Configure Action
       2.1 Pre-requisite 
           A Solution should be created.
           Please refer below section on how to create a solution:-
           Solution-> Business Flows                                  

       2.2 How to create this Action
            2.2.1 Check File Exists

              1)  Double click on the Business flow under Buisness Flow tab.  Click on ‘Add New’ button and click on Actions Library. 

                                

                         

                      2)  Go to the ‘General Actions’ tab and select the ‘File Operations’ then click on 'Add to Actions' button shown as below.

                                  

                      


                      3)   Double click on the Action to open it in Edit mode. Below screen should be displayed.                                                                                        The Operation type ‘CheckFileExists’ is displayed by default.
                            Select a file using the Browse button. 

                                

                4)    Run Action will check if the given file exists in the given folder. The action fails if the file is not found.
      
            2.2.2    Other Operation Types

                                        
                          

                       Force Copy  -  If for some reason, a file is not getting copies then Force copy can be used.

                       Move :-

                           Select ‘Move’ option from the Operations Type drop down.
                           Select the ‘Source File Path:’ using the Browse button. 
                           Enter the Destination Path.                     

                      Run command
                       This is same as Execute command. The only difference is that Execute command returns an error message when a file is not 
                        found whereas the Run command gives a proper informative message when a file is not found.
