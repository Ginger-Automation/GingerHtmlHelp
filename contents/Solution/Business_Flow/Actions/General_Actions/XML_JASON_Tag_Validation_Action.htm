XML/JASON Tag Validation Action

How to Configure Action         
    

1)  Double click on the Business flow under Buisness Flow tab.  Click on ‘Add New’ button and click on Actions Library. 

                                

    2)   Go to the ‘General Actions’ tab and select the ‘XML Tag Validation Action’ then click on 'Add to Actions' button shown as below. 
                                          

    3)     Double click on the Action to open it in Edit mode.  Below screen should be displayed.
             •    XML Template File can be given using the Browse button. To use this, the ‘Take Request from file Path’ option has to be checked.
             •    The attributes can be added using the ‘Add New’ button as shown below..
             
                         

                              

 

               When the ‘Take Request from file Path’ option is not checked then the user needs to write the xml in the "Template File Content" field.
               


         4)    The Run Action button will validate the given Tags.
