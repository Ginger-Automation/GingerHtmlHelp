XML/JASON Tag Validation Action

How to Configure Action
          
    1)    Go to Automate tab. In the Activity Actions, click on ‘Add New’ button
               

    2)    Go to the ‘General Actions’ tab and select the ‘XML Tag Validation Action’ then click on Add Action button given in the below window
                                          

    3)    Below screen should be displayed.
             •    XML Template File can be given using the Browse button. To use this, the ‘Take Request from file Path’ option has to be checked.
             •    The attributes can be added using the ‘Add New’ button as shown below..
             •    Click on Ok button to add the action.
                                      
             
    4)    The action gets added in Activity Action list
                


    5)    The Run Action button will validate the given Tags.
