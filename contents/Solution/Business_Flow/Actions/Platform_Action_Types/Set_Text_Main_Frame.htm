Set Text Main Frame

Pre-requisite 
     A Solution should be created with Agent having Main Frame Platform.
                          
Action Operation                    

         1)  Double click on the Business flow under Buisness Flow tab.  Click on ‘Add New’ button and click on Actions Library. 

                                

 
             2)   Go to the ‘Platform Actions’ tab and select the ‘Set Text MainFrame’ then click on 'Add to Actions' button shown as below.
   
                                                   


              3)   Double click on the Action to open it in Edit mode. Below screen should be displayed.  
                     •    ‘Locate By:’ drop down can be used to indicate the locator.
                     •    The ‘Locate Value’ can be used using the ‘Open Window Explorer to select Element’ button.
                           Refer the ‘Locate by-Locate Value’ document to know how to use this.

                     •    Select the ‘Set Text Mode’ from the drop down.
                     •    ‘Set Text and Send Enter’ check is box is used as required.
                     •    ‘Value’ is entered if required either manually or using a variable.
                    Click on the Run Action button to execute the Action.

                   The ‘Set Text Mode’ drop down has the below options:-
                   Set  Single Field (default selected)—used to work on single field
                   Set  Multiple Fields                    ------used to work on multiple fields
