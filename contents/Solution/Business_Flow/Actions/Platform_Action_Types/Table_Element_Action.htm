Table Element Action

Pre-requisite 
    A Solution should be created with Agent having Web or Mobile Platform.
                          
Action Operation
                      
   1.    Double click on the Business flow. In the Activity Actions, click on ‘Add New’ button               

          

 
   2.    In the ‘Platform Actions’ tab, select the ‘Table Element Action’ then click on ‘Add Action’ button given in the below window
                                  

   3.    Below screen should be displayed.  
      •    ‘Locate By:’ drop down can be used to indicate the locator.
         The ‘Locate Value’ can be used using the ‘Open Window Explorer to select Element’ button.
         Refer the Locate By-Locate Value document for how to use it.

      •    After locating the element, its columns will be populated the Column drop down with either Title or number.
      •    Row can be selected as per requirement. The ‘Where’ option will populate fields to enter the condition
      •    Select  Operation has multiple options :- ActivateRow, Async  click, click, double clikc, Get value, Get row count,

           Get selected row, IseCell enabled, set value, Set Focus, Toggle, Type and Winclick.
      •    ‘Value’ can be added if required.
      •    Click on Ok to add the action to the Activity Actions.

             


            The ‘Operation Type’ have the below options:-
            Close Window- Used to close a window
            Clear              ------ used to make the text box blank by clearing the contents
            Get Font      -------- used to get the Font of the text box  
            Get Height   -------  used to get the height of the text box 
            Get Input Length – used to get the length of the text box
            Get Style          ---- used to get the style of the text box
            Get Value        ----- used to get the content in the text box
            Get Width      ------ used to get the width of the text box
            Is Disabled       ---- used to check if the text box is disabled
            Is Displayed    ----- used to check if the text box is displayed
            Is Prepopulated  -- -used to check if anything is prepopulated in the text box
            Is Required        --- used to check if the text box is required
            Set Focus         ---- used to set the focus on the text box
            Set Value          ---- used to enter a value in the text box
            Set Value Fast  ---- used to enter a value in the text box in a quick way
