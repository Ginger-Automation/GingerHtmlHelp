Smart Syn Action

Pre-requisite 
    A Solution should be created with Agent having Web, Windows, PowerBuilder or Java Platform.
                          
Action Operation
                      
1)    Double click on the Business flow. In the Activity Actions, click on ‘Add New’ button
            
 

2)    In the ‘Platform Actions’ tab, select the ‘Smart Sync Action’ then click on ‘Add Action’ button given in the below window
                                              


3)    Below screen should be displayed.  
       ‘Locate By:’ drop down can be used to indicate the locator.
       The ‘Locate Value’ can be used using the ‘Open Window Explorer to select Element’ button.
       Refer the ‘Locate by-Locate Value’ document to know how to use this.

       ‘Action Type’ has 2 options :-
          a)    ‘WaitUntilDisappear’ – this is used when it is required to wait until a screen disappears say after processing some data
         b)    ‘WaitUntilDisplay’ – this is used when it is required to wait until a screen is displayed

                ‘WaitTime’  is used to enter the time required to Sync the windows, eg. 10 (sec)
                 Click on Ok to add the action to the Activity Actions.
