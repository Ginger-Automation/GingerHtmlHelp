What is an Agent?

Agents are the drivers which are used for recording or execution for the target application under test.  

How to create an Agent?
   i)    When a solution is created, an agent specific to the platform is automatically added.
  ii)    To add another agent, Configurations Tab-> Agents. Click on the ‘+’ button to add a new agent as shown below.
          

 

  iii)    Add New Agent wizard is displayed.

          Clicking on Next button displayed the Agent details page where the platform and driver can be selected.

          

 

          Platform and Driver types:-
          •    Android Device – used for testing on devices which are on android platforms. 
                                             It has AndroidADB driver.         
          •    DOS                  – used for applications which are on DOC platforms. 
                                             It has DOS console driver.
          •    JAVA                  – used for applications which are on Java platforms. 
                                              It has Java driver.
          •    MainFrame         - used for applications which are on Main Frame.  
                                              It has Main Frame 3270 driver.
          •    Mobile                - used for Mobile applications. It has below driver types:-
                      i)    Mobile Appium  Android
                     ii)    Mobile Appium IOS
                    iii)    PerfectoMobile
                    iv)    Mobile Appium Android Browser
                     v)    Mobile Appium IOS Browser


          •    Power Builder    - used for applications which are on Power Builder. 
                                             It has Power Builder driver and FlaUIPB driver.
          •    Unix                    - used for applications which are on Unix. It has UnixShell driver.
          •    VBScript             - used for applications which are on VBScript.
          •    Web                     - used for applications which are on web platform. It has below driver types:-
                     i)    Selenium Chrome
                    ii)    Selenium Firefox
                   iii)    Selenium IE
                   iv)    Selenium Remote Web Driver
                    v)    Selenium Edge
                       
          •    WebServices       - used for applications which are using web services.
                                              It has WebServices driver.
          •    Windows             - used for applications which are on Windows.
                                              It has Windows Automation driver and FlaUIwindow driver.
