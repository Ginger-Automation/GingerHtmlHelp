Mobile iOS Testing Device Setup

2.1. Setting Device Configurations 

      a.    Connect the device to the Mac via USB
      b.    When connecting the iOS device to the Mac machine you will be asked (on the device) if to trust the connected Mac- select “Trust”
      c.    For iOS 6.0 and up Turn on “Web inspector” on iOS device (settings > safari > advanced)
      d.    For iOS 8.0 and up Turn on "UI Automation" toggle on iOS device (settings > Developer)
      
2.2.    Getting Device Details

      a.    Go to the Mac main menu bar -> “Utilities” -> “Terminal”
      b.    Type and run below command:
            instruments –s devices
      c.    You will see list of all known devices, the list contains the Simulators and real devices (which connected to the Mac via USB)- those

are the devices which will be known by Appium and automation will be available for them
      d.    For each device there are 3 important details
      a.    Device Name
      b.    Platform Version
      c.    Device ID (UDID)
             
      e.    The device details will be used for the Ginger Agent configuration so automation will run on the relevant device
