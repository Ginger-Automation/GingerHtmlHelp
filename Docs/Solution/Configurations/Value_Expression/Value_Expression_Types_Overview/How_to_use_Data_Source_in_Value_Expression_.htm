How to use Data Source in Value Expression?

      Expand the Data Sources by double clicking on it until you see the tables in it.  

 

a. Key Table

The Key table will be displayed for selection.

The Table data displays all the values in the table so that the user does not have to go the Data source to see the Table Data.

Only the Key Name column is available for selection.

 

Once the Key name is selected, user can click the ‘Test Value Expression’ as shown below to check the value it displays.   

 

b. Customized Table

The Table Data displays the content in the table.

As shown below, the cell identifier can be ‘By Select Cell’, ‘By Query’ or by ‘Customized’

For customized, User can select the column name from the drop down.

The row can be selected by ‘Row Number’, ‘Next available row’ (this is mainly used to add values in the table)

Or the Where clause.

 

In the Where Clause, user can add the desired column, operator and the value. Click on the Ok button.

 

Click on the ‘Test Value Expression’ button gives the value that will be evaluated and used when the Action will be run.
