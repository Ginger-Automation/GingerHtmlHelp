Script Action

How to Configure Action

    1)    Go to Automate tab. In the Activity Actions, click on ‘Add New’ button
               

    2)    Go to the ‘General Actions’ tab and select the ‘Script Action’ then click on ‘Add Action’ button given in the below window
                           

    3)    Below screen should be displayed.  
           The ‘Locate Value’ can be used using the ‘Open Window Explorer to select Element’ button shown below to locate the element where scripting will be used.
            The ‘Locate By:’ drop down values gets auto-populated when the Open Window Explorer to select Element’ button is used.
            ‘Script Interpreter’ has default option as ‘VBS’. Other options are ‘JS’ and ‘Other’.
             ‘Value’ can be used if required.
           


    4)    Clicking on ‘Open Window Explorer to select Element’ button opens the ‘Ginger Window Explorer’ window. 
            Web agent should be opened in order to spy the element.
            Click on the ‘Live Spy’ button to Hover with the mouse over the Element you want to spy and Click/Hold Down ‘Ctrl’ Key in the Web Agent.
             


    5)    The below screen is displayed with highlighted element when the spy element is used.
              


    6)    The ‘Actions’ tab in the ‘Ginger Window Explorer’ gets updated with the Element identifiers. 
            Select one of the identifier eg. ByID in the ‘Locators’ section.
            The ‘Actions’ section shows the list of Actions that can be performed on the element. Choose one of them.
            Use the button ‘Add Selected Action to Business Flow’ as shown below to add this Action. 

              

 
    7)    As shown in the below screen the ‘Locate By:’ and ‘Locate Value:’ get populated with the selected values.
           Click on Ok button to add the action in the ‘Activity Actions’.

            The Locate By drop down has multiple options.
             


       

    8)    Run Action will execute the Script Action.
