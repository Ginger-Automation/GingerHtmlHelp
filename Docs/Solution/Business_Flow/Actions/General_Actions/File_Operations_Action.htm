File Operations Action

1.  Introduction    
     File Operations Action is used to various operations on a file. 
     Below is the available list of operations:-
        1)    Check File exists
        2)    Check folder Exists
        3)    Copy
        4)    Execute
        5)    Force Copy
        6)    Move
        7)    Run command
        
2.    How to Configure Action
       2.1 Pre-requisite 
           A Solution should be created.
           Please refer below section on how to create a solution:-
           Solution-> Business Flows                                  

       2.2 How to create this Action
            2.2.1 Check File Exists

                1)    Go to Automate tab. In the Activity Actions, click on ‘Add New’ button
                            


                2)    Go to the ‘General Actions’ tab and select the ‘File Operations’ then click on ‘Add Action’ button given in the below window                       

                      


                3)    Below screen should be displayed.  The Operation type ‘checkfileExists’ is displayed by default.
                       Select a file using the Browse button. Click on Ok button to add the Action to the Activity Actions.

                          

                4)    Run Action will check if the given file exists in the given folder. The action fails if the file is not found.
      
            2.2.2    Check folder Exists

                1)    Follow steps 1 & 2 from the Check File Exists action.

                2)    Below screen should be displayed. Select ‘CheckFolderExists’ from the Operation Type drop down.
                       Select the folder path using the Browse button.
                       Click on Ok button to add the action to the Activity Actions. 
                       

                3)    The Run Action will check if the given folder exists in the given path. The action fails if the folder is not found.


            2.2.3   Copy

                1)    Follow steps 1 & 2 from the Check File Exists action.


                2)    Below screen is displayed. Select ‘Copy’ option from the Operations Type drop down.
                       Select the ‘Source File Path:’ using the Browse button.
                       Enter the ‘Destination Path (Folder/File)’ in the given field.
                       Click on Ok button to add the action to the Activity Actions.

                        

                3)    The Run Action will Copy the given file in the Destination folder/file.

            2.2.4    Execute

                1)    Follow steps 1 & 2 from the Check File Exists action.

                2)    Below screen is displayed. Select ‘Execute’ option from the Operations Type drop down.
                       Select the ‘Source File Path:’ using the Browse button. 
                       Click on Ok button to add the action to the Activity Actions.

                      
 
                3)    The Run Action executes the file that was selected.

            2.2.5    Force Copy

                      If for some reason, a file is not getting copies then Force copy can be used.

                       1)    Follow steps 1 & 2 from the Check File Exists action.

                       2)    Below screen is displayed. Select ‘Force Copy’ option from the Operations Type drop down.
                              Select the ‘Source File Path:’ using the Browse button. 
                              Enter the Destination Path Folder/File.
                             Click on Ok button to add the action to the Activity Actions.

                               

                    3)    The Run Action copies the given file into the destination file.

            2.2.6    Move

                    1)    Follow steps 1 & 2 from the Check File Exists action.

                    2)    Below screen is displayed. Select ‘Move’ option from the Operations Type drop down.
                           Select the ‘Source File Path:’ using the Browse button. 
                           Enter the Destination Path.
                           Click on Ok button to add the action to the Activity Actions.

                             
 
                    3)    The Run Action movies the given file into the destination path.

            2.2.7    Run command
                       This is same as Execute command. The only difference is that Execute command returns an error message when a file is not 
                        found whereas the Run command gives a proper informative message when a file is not found.
