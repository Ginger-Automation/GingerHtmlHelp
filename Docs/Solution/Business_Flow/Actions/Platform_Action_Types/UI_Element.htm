UI Element

Pre-requisite:-
    A Solution should be created having Agent as Web, Mobile, Java, AndroidDevice, PowerBuilder platform.
                          
1. Action Operation:-
                      
   i)    Double click on the Business flow. In the Activity Actions, click on ‘Add New’ button
                  


  ii)    In the ‘Platform Actions’ tab, select the ‘UIElement Action’ then click on ‘Add Action’ button given in the below window                                           


  iii)    Below screen should be displayed.  
        Details on Locate By-Locate Value can be found in Platform Actions.

        Locate value can be manually entered or a variable can be selected using the ‘…’ button.
        Element Type and Element Action needs to be selected from the drop down.

     

2.    Action Operation for setting Password

    In the below screen enter the details on Locate By-Locate Value.
    Select the Element Type as Text box.
    Select Element Action as ‘Send Keys’.  Enter the password value. User can create a variable with type as Password Variable and use it in the Value field. Click on the’…’ button near the Value field to add the variable.
    On run time the password will be entered with display of ‘*’ instead of displaying the actual alphabets.

    


3.    Action Operation- Submit Action

    In the below screen enter the details on Locate By-Locate Value.
    Select the Element Type as ‘Button’.
    Select Element Action as ‘Submit’.  
     


4.    Action Operation- Text Box Action

    In the below screen enter the details on Locate By-Locate Value.
    Select the Element Type as ‘Text Box’.
    Select Element Action as ‘Send Keys’.  

    

    Below are the Element Actions used more often:-
    -    Clear Value
    -    Get Value
    -    Is Disabled
    -    Get Font
    -    Is Value Populated
    -    Is Visible
    -    GetTextLength

5.    How to use Page Object Model in the Action

    i.    Add the Action and in the Locate By select ‘Page Objects Model Element’ option.
    ii.    In the Locate Value, select the desired Page Object Model using the pen icon. 

       
                    

    iii.    Select the desired element and click on the ‘Select Element’ button.

      

    iv.    The Element type is automatically populated. User can then select the desired Element Action for it.

    


    v.    User can also check where is the element exactly located in the page by clicking on the ‘Highlight Element’ button as shown below.
        For this the Agent should be running and the page should already be displayed in it.
