Screenshot Action

How to Configure Action
      1. Pre-requisite 
              A Solution should be created with Agent having Web, Mobile, Windows, PowerBuilder or Andriod Device Platform.
                          
      2. Action Operation
                      
           1)    Double click on the Business flow. In the Activity Actions, click on ‘Add New’ button

                     

              2) In the ‘Platform Actions’ tab, select the ‘Screenshot Action’ then click on ‘Add Action’ button given in the below window

               

 

              3) Below screen should be displayed. 

     ‘Save to folder Name’ is used to enter the folder path which is selected using ‘Browser’ button.

     Click on Ok to add the action to the Activity Actions.

      

4) Run Action will take screenshot of the active window in the Web Agent and save it to the given folder.

     The screenshot name will be <Date><Time>Screen Shot Action.jpg   eg. ‘12_06_2017_15_46_03_380Screen Shot Action.jpg’
