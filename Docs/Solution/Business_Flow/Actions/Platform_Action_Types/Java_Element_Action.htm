Java Element Action

How to Configure Action
      1. Pre-requisite 
                         A Solution should be created with Agent having Java Platform.
                          
      2. Action Operation
                      
           1)    Double click on the Business flow. In the Activity Actions, click on ‘Add New’ button
                   

 

           2)    Below example is created using the Browse Action. 
                   In the ‘Platform Actions’ tab, select the ‘Java Element Action’ then click on ‘Add Action’ button given in the below window

                   

              
           3)    Below screen should be displayed.  
                   •    ‘Operation to Perform’ drop down can be used to indicate to perform the action.
                   •    ‘Wait for Idle Option:’ can also be given.
                   •    ‘Value’ can be also be added

                   


            4)    Run action will perform the given action.
