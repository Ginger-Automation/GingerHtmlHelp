Ginger Work Flow

  1. Click on ‘Create New Solution’ Tab as highlighted below ( + Sign) button from the drop down.

                           

              

           Enter the details in the screen as given below. Platform is the one on which your application runs.

           The ‘Create’ button will load the solution

                         

                                                                                                                                                             

  1. ‘Business Flow 1’ will be by default added to the ‘Business Flow’. 

                     

   

  1. Double click on the Business Flow 1 tab or click on the ‘Automate’ button in the right panel.

          The ‘Activity1’ will be added by default in the ‘Activity’ section

             

  1. In the ‘Activity-Actions’ section, you can add the actions that are required to execute the solution.

              

  1. Go to Configurations-> Agents and enter the details. Eg. for Web Platform, select the driver type which can be IE, Chrome, etc. For Unix platform, the unix connection details has to be entered.
