Support

 1.    Support Site    
       Expand the Support option as shown below.

       
          


         Click on the Support Site option. It opens the Ginger support site

           
          
     1.1   Ginger Installers
           The section has Official, Beta and Alpha releases of Ginger.


           Official release has many new features and bug fixes. Hence a thorough testing is done on this installer before releasing it.

           Beta release is available on every Monday for various bug fixes. Basic functionality testing is done on this before releasing it.

           Alpha release is available as per the urgent fixes required by projects. No testing is done on this release.
            

 

     1.2   Other Downloads
 
           This has plugins and other configurations specific to projects.

           
                          

 2.    Open Ticket    
      Tickets on the Ginger core team, for support issues, can be opened from here.
        


    Existing tickets can be searched from the Support site. 
    Tickets are closed from here only by changing the status of the ticket.
    


 3.    Q&A Forum    
 
    There can be general queries which need require tickets to be opened. 
    Such queries can be asked on this forum, like why the Database action is failing with 
    error as Oracle.ManagedDataAccess when Database action is run. Answers to such 
    general questions are given by the support and core team members on this forum. 
    Questions & Answers opened by others can also be viewed here.

    

   


 4.    Help Library    

    A list of support documents and videos of Ginger are displayed here. Search helps in searching the content.

    It can also be opened by pressing F1 on the content whose help you desire.
