How to Sync/Unsync Runners?

The below button indicates Sync option which is selected by default.

                                            

           

            As shown below, the execution of Business Flow/Activity/Action which are highlighted are

           only going on.

           The user will not be able to highlight to any other Business

            Flow/Activity/Action which are not under execution.

            This helps to know which Business Flow/Activity/Action is under execution currently.

            

 

 

              The below button indicates  Un-Sync option.

           

 

           

                As shown below, the user can highlight any Business Flow/Activity/Action irrespective of which Business Flow/Activity/Action execution is going on.
