How to get latest solution from SVN?

1)    The latest updates of the solution can be fetched by clicking on the ‘Source Control Operations’ button as shown below. Select ‘Get Latest Solution Changes’ option. 
        Whatever changes are updated by other users and stored in the SVN will be downloaded.

        


        
        A message is displayed once the download is successful giving the information the revision number.
                   

 

    2) You can also get the updated changes from SVN only for a particular component. 
         Right click on the component, Go to Source Control and click “Get Latest Updates” option.
