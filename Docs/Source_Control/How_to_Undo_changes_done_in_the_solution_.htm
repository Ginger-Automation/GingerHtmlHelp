How to Undo changes done in the solution?

When changes are made in the solution and you do not want that changes to be saved in the SVN then you can Undo the changes.
 This will take your Solution back to the latest version from the SVN. The changes made on the local PC will be lost.

    1)    Right click on the component whose changes you want to remove and select the “Undo Changes” option.

           

 

    2)    A message of confirm is displayed. After clicking the Yes button, the changes will not be available in the solution.
