Export/ Import to Excel

1.    Export Data Source to Excel

     1)    Right click on the Data Source and select the option to Export to Excel as shown below.

             

           
     2)    Select the File name and click on ok button. (User need to first create an empty Excel file to use it).

            


            The Excel file will have separate sheet for every table with the Table name as the sheet name.
                         


2.    Export Table to Excel (Customized or Key table)

     1)    Select the Export to Excel option by right clicking on the desired table.

            

     2)    Select the File and the sheet where you want to export and click on Ok button to export the data.

            

3.    Import Data Source from Excel 

     1)    Right click on the Data Source and select the Import from Excel option

            

     2)    In the Wizard window, click on the Next button after the Information of Data source is displayed.
           Then select the desired excel file and click on Next button.

           
 
     3)    Select the desired Sheet in the file

            

            a)    When user selects ‘Sheet as first row as column name’ then the same column name is given to the column in the table.                                      Field 1 etc is not added in the Table.

                    Columns without the ‘Sheet as first row as column name’ option
                     

                    Columns with the ‘Sheet as first row as column name’ option

                     
 
           b)    When ‘Is the Model Parameter File’ is selected then the first column of the sheet is taken as the fields of the table and the rest                         are  considered as their values.
                    
 

     4)    The data from the Excel file is displayed
                                      

     5)    User can filter the desired data if required. 
             Use the ‘Select Row(s) Where’ filed to enter filter and view it using  ‘View Filtered Data’ button.
             If the column has space in it then use square brackets eg. [Parameter Name]=’ABC’

     6)    The Imported Table is given the name of the sheet in the file.
