Data Source & Table Manipulation Options

1.    Data Source Manipulation Options 
       We have various options available to work and manipulate data in the Data Source. All the below manipulation options can be found in:
         •    Pop up Menu by doing Right Click on the Data Sources node.
         •    Data Source Page Toolbar
         •    Explorer Toolbar 

          

 

2.    Data Source Table Manipulation Options 
       We have various options available to work and manipulate data in the Data Source. All the below manipulation options can be found in:
         •    Data Source Table Menu.
         •    Data Source Table Page Toolbar
         •    Explorer Toolbar 
           


         2.1    Commit - Save Al Local changes of the Table to Database (MDB & LiteDB File).

         2.2    Rename - Rename the Table.

         2.3    Refresh - Refresh the Table data in the Data Source Table Data Grid. It will discard all local changes.

         2.4    Duplicate - Create a copy of the Current Table.(Not available in Data Source Table Page toolbar)

         2.5    Delete - Delete Current Table.

         2.6    Add Column - Add a new Column to the table. It will save all local changes. (Only available in Data Source Table Page toolbar)

         2.7    Remove Column - Remove Column from table. (Only available in Data Source Table Page toolbar)
