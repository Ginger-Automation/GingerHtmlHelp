How to Export optional values to Excel or Data Source?

1. How to Export optional values to an Excel file?

    a.    Select the desired API and go to the Model Parameters tab.
    b.    Click on the ‘Export Parameters to Excel File’ button.
            


    c.    A message is displayed on successful export.

          


             
    d.    The file is opened on clicking the Ok button of the above message.
           This file is stored in the Documents folder of the local PC.

           

2. How to Export optional values to Data Source?

    a.    Select the desired API and go to the Model Parameters tab.
    b.    Click on the ‘Export Parameters to DataSource’ button.
          
                            
    c.    Data Source tables are displayed in tree format. Parameters can be exported to only customized table 
          and hence only the customized table in the data source are displayed in the window.
          Click on the desired table and then click on the Select button.

           

 

    d.    A message is displayed on successful export.

            

    e.    Go to Resources-> Data Sources. Right click on the GingerDataSource node and select refresh button. 
          The exported data is displayed after the refresh is done.
