How to update Shared Repository with modified Item from Solution?

•    Open the Item in the Solution in Edit mode by double clicking on it.
     The start in orange color as shown below indicates that the item is from shared repository.

       


•    Make the desired changes and hover the mouse over the orange star.
     Two icons (pin and star+) are displayed

     Click on the star+ as shown below to update the changes in the shared repository

       

 
•    Below message is displayed to confirm if you want to override the item in the shared repository.
    Clicking on Yes button saves the changes in shared repository.
    Clicking on No button does not override the item in shared repository.

      
 

•    Clicking on Yes button updates the shared repository. 
    A back up of the existing item are saved in the ‘PreVersions’ folder as shown below.

     

•    Shared Repository Items can also be edited from the Shared Repository tab
