How to update the Shared Repository item instances in the Solution with latest updates from shared repository?

  • Update the item in the shared repository.

  • As shown in above point no.7 , open the ‘Repository Item Usage window’ to get the usages in the solution.

  • Select the desired items and click on the ‘Update All Selected’ or use ‘Save All Updated Usages’ button.

This updates the selected instances in the Solution as per changes made in shared repository.
