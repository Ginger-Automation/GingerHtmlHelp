Configure ALM Items Fields

Ginger allows to configure the ALM Test Case and Test Set items fields’ value to be updated when doing export from Ginger to ALM.

For configuring the items fields values:
1.    Click on “ALM” button from the menu

      
2.    Select “ALM Items Fields Configuration”
3.    “ALM Items Fields Configuration” window will open
4.    Click on “Refresh” button (near the search) in order to refresh ALM Items Fields Configurations from ALM.
    Note that Refreshing ALM Items Fields Configurations from RQM ALM can take several hours, you can close “ALM Items Fields Configuration” during that process, but the computer and Ginger software must stay open.


 


5.    For fields you want their value to be updated during export:
a.    Check the “To Update” checkbox
b.    Select value from list or set the value (in case of free text field) in “Selected Value” column
6.    Click on “Save” button to save the configurations. 
   Note:
   •    Only fields which “To Update” field will be checked for will be saved
   •    Fields which are “Mandatory” will be updated during the export process even if they will be un-checked (with default value)
