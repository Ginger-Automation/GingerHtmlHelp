Export Business Flow to ALM

Ginger allows to export a Ginger Business Flow to ALM.
The export is been done in 2 parts:
     1.    Export Activities Groups to ALM Test Cases- each one of the exported Business Flow Activities Groups will be exported to an ALM Test Case item in ALM Test Plan.
    2.    Export Business Flow to ALM Test Set- an ALM
         a.    QC ALM Export - Test Set item will be created in QC ALM Test Lab into user selected folder.
         b.    RQM ALM export - Test Plan will be exported to RQM, there are no folders in RQM ALM.
    3.    all of the created ALM Test Cases from step 1 will be added to the Business Flow to represent the execution flow

Steps to export Business Flow:
1.    Launch the export process by:

    a. Double click the Business Flow to go to automation area. Click on the “Export to ALM” button      

          

            

    b.    Right click the Business Flows to export and select the “Export All to ALM” option. It will export all the Business Flows of the solution to ALM
         

    c.    Right click on the desired Business Flow in the Solution Tree and select “Export to ALM”

         

 
2.    In case ALM connection details are missing or incorrect you will be asked to connect to ALM:
       


3.    Exporting to RQM or QC ALM – Ginger will try to use the current saved configurations saved on step 3 and use them to export to selected

     ALM type. 
     Exporting to QC ALM - if QC ALM has been set in step 3, Ginger will use saved configurations and export to QC ALM:
     In case not all of the exported Business Flow Activities Groups were exported before to ALM:


         a. ALM Test Plan browser window will be shown and you will need to select to which ALM Test Plan folder the Activities Groups will be exported as ALM Test Cases
             


        b. Select a folder and click on “Select Folder” button
        c. All Activities Groups which were not exported before will be exported into the selected folder as ALM Test Cases
               

        d. For each Activities Group the details which will be exported are:
             i.    Activities Group Name
            ii.    For each Activity in the Activities Group an ALM Step item will be created containing the:
                •    Activity Name
                •    Activity Description
                •    Activity list of Variables Names
                •    Activity list of Actions Description
                •    Activity Expected 
               

            

4.    ALM Test Lab browser window will be shown for selecting the ALM Test Lab folder to export the Business Flow ALM Test Set into

      a.    Select an ALM Test Lab folder
      b.    Click on “Select Folder” button
           

5.    The Business Flow will be exported to ALM- new Test Set will be created with the Business Name and all the matching Test Cases for the Business Flow Activities Groups will be added into it Execution Flow
           

 
6.    In case running from Automate tab you will be asked to save the Business Flow for saving the links to the ALM items (Test Cases & Test Set) 
7.    Exporting to RQM ALM: if RQM ALM has been set in step 3, Ginger will use saved configurations and export to RQM ALM:
8.    RQM ALM export is similar to QC ALM export, the only difference is that in RQM there are no folders and when exporting from Ginger to RQM there will be no option to select where to export to.

        Note: if the Test Plan already exist in RQM ALM, exporting it again will update the existing Test Plan and wont create another instance.

         Note: 
           •    Each business must have at least  one Activity Group
           •    It is possible to Export the Business Flow Activities Groups separately by:
                  a.    Clicking on the Export icon which placed on the “Activities Groups” page after selecting all the Activities Group to export
                         


                  b.    Right click on Shared Activities Group item in the Solution tree and select “Export to ALM”. For exporting group of Activities Groups you can right click on a folder of Activities Groups and select “Export All to ALM”
